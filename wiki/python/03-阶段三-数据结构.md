# 阶段三：数据结构（第 5-6 周）

> 目标：掌握 Python 四大内置数据结构，能用推导式写出 Pythonic 代码

---

## 1. 列表 list

```python
# 创建
nums = [1, 2, 3, 4, 5]
mixed = [1, "hello", True, None]  # 可以混类型

# 增删改查
nums.append(6)           # 末尾追加
nums.insert(0, 0)        # 指定位置插入
nums.remove(3)           # 删除第一个匹配值
nums.pop()               # 删除并返回末尾元素
nums.pop(0)              # 删除并返回指定位置
del nums[1]              # 删除指定位置

# 切片（Python 最强大的特性之一）
nums = [0, 1, 2, 3, 4, 5]
nums[1:4]       # [1, 2, 3]（左闭右开）
nums[:3]        # [0, 1, 2]（从头开始）
nums[3:]        # [3, 4, 5]（到末尾）
nums[::2]       # [0, 2, 4]（步长 2）
nums[::-1]      # [5, 4, 3, 2, 1, 0]（反转）

# 排序
nums.sort()              # 原地排序
nums.sort(reverse=True)  # 降序
sorted(nums)             # 返回新列表，不修改原列表

# 其他
len(nums)        # 长度
3 in nums        # True（成员检查）
nums.count(3)    # 出现次数
nums.index(3)    # 第一次出现的位置
```

**Java 类比**：`list` ≈ `ArrayList`，但更灵活。切片是 Python 独有，Java 需要 `subList()`。

---

## 2. 字典 dict

```python
# 创建
user = {"name": "Alice", "age": 30}
user = dict(name="Alice", age=30)  # 另一种写法

# 增删改查
user["email"] = "alice@example.com"  # 新增/修改
user.get("phone", "N/A")             # 安全取值，不存在返回默认值
del user["email"]                     # 删除
user.pop("email", None)              # 删除并返回，不存在返回 None

# 遍历
for key in user:                      # 遍历 key
    print(key)

for key, value in user.items():       # 遍历 key-value
    print(f"{key}: {value}")

for value in user.values():           # 遍历 value
    print(value)

# 合并
a = {"x": 1}
b = {"y": 2}
c = {**a, **b}       # {"x": 1, "y": 2}（Python 3.5+）
c = a | b            # {"x": 1, "y": 2}（Python 3.9+）
```

**Java 类比**：`dict` ≈ `HashMap`。`get(key, default)` 比 Java 的 `getOrDefault()` 更早出现。

---

## 3. 集合 set

```python
# 创建
s = {1, 2, 3}
s = set([1, 2, 2, 3])   # {1, 2, 3}（自动去重）

# 运算
a = {1, 2, 3}
b = {2, 3, 4}
a & b          # {2, 3}（交集）
a | b          # {1, 2, 3, 4}（并集）
a - b          # {1}（差集）
a ^ b          # {1, 4}（对称差集）

# 成员检查（O(1)，比 list 快）
1 in a         # True
```

**Java 类比**：`set` ≈ `HashSet`，运算符对应 `retainAll`/`addAll`/`removeAll`。

---

## 4. 元组 tuple

```python
# 不可变列表
t = (1, 2, 3)
t[0]           # 1
# t[0] = 99    # TypeError! 不可修改

# 解包
x, y, z = t    # x=1, y=2, z=3

# 命名元组（可读性更好）
from collections import namedtuple
Point = namedtuple("Point", ["x", "y"])
p = Point(3, 4)
p.x            # 3
p.y            # 4
```

**Java 类比**：`tuple` ≈ 没有直接对应，类似 `record` 但无字段名。命名元组 ≈ `record Point(int x, int y)`。

## 5. 推导式

```python
# 列表推导式（最常用）
squares = [x**2 for x in range(10)]            # [0, 1, 4, 9, ...]
evens = [x for x in range(20) if x % 2 == 0]  # [0, 2, 4, ...]

# 字典推导式
word_lengths = {w: len(w) for w in ["hello", "world"]}  # {"hello": 5, "world": 5}

# 集合推导式
unique_lengths = {len(w) for w in ["hello", "world", "hi"]}  # {2, 5}

# 嵌套推导式
matrix = [[1, 2], [3, 4], [5, 6]]
flat = [x for row in matrix for x in row]  # [1, 2, 3, 4, 5, 6]
```

**Java 类比**：推导式 ≈ Stream API。`[x**2 for x in nums if x > 0]` ≈ `nums.stream().filter(x -> x > 0).map(x -> x*x).collect(toList())`。

## 6. 常用内置函数

```python
nums = [3, 1, 4, 1, 5, 9]

len(nums)        # 6（长度）
min(nums)        # 1
max(nums)        # 9
sum(nums)        # 23
sorted(nums)     # [1, 1, 3, 4, 5, 9]

# zip（并行遍历）
names = ["Alice", "Bob"]
ages = [30, 25]
for name, age in zip(names, ages):
    print(f"{name}: {age}")

# any / all
any([0, 0, 1])   # True（至少一个为真）
all([1, 2, 3])   # True（全部为真）
```
