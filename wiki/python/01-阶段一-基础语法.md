# 阶段一：基础语法（第 1-2 周）

> 目标：能写 Python 脚本，理解基本类型和运算

---

## 1. 环境搭建

```bash
# macOS 安装 pyenv
brew install pyenv
pyenv install 3.12.4
pyenv global 3.12.4

# 验证
python --version  # Python 3.12.4

# 创建虚拟环境（每个项目独立依赖）
python -m venv .venv
source .venv/bin/activate
pip install requests  # 安装依赖
```

**Java 类比**：pyenv ≈ SDKMAN，venv ≈ Maven 的 project-local dependency，pip ≈ Maven Central。

---

## 2. 变量与类型

```python
# 动态类型，无需声明
name = "Alice"        # str
age = 30              # int
height = 1.75         # float
is_active = True      # bool
data = None           # NoneType（类似 Java null）
```

| Python | Java | 说明 |
|---|---|---|
| `int` | `long` | Python 整数无上限 |
| `float` | `double` | 64 位浮点 |
| `str` | `String` | Python 字符串不可变 |
| `bool` | `boolean` | `True`/`False`（大写开头） |
| `None` | `null` | 用 `is None` 判断，不用 `==` |

```python
# 类型检查
type(name)        # <class 'str'>
isinstance(age, int)  # True

# 类型标注（Python 3.5+，不强制，但推荐）
age: int = 30
```

---

## 3. 字符串

```python
# 三种引号
s1 = 'hello'
s2 = "hello"
s3 = """多行
字符串"""

# f-string（Python 3.6+，最常用）
name = "Alice"
f"Hello, {name}"           # "Hello, Alice"
f"Next year: {age + 1}"    # 表达式直接嵌入

# 常用方法
"hello".upper()        # "HELLO"
"HELLO".lower()        # "hello"
" a,b,c ".split(",")   # [' a', 'b', 'c ']
" a,b,c ".strip()      # "a,b,c"
"hello".replace("l", "L")  # "heLLo"
",".join(["a", "b", "c"])  # "a,b,c"
```

**Java 类比**：f-string ≈ `String.format()`，但更简洁。Python 字符串方法直接在对象上调用，不需要 `StringUtils` 工具类。

---

## 4. 运算符

```python
# 算术
10 / 3    # 3.333...（Python 3 除法返回 float）
10 // 3   # 3（整除）
10 % 3    # 1（取模）
2 ** 10   # 1024（幂运算）

# 比较（和 Java 一样）
==  !=  <  >  <=  >=

# 逻辑（用英文单词，不是符号）
and  or  not

# 身份 vs 相等
a = [1, 2, 3]
b = [1, 2, 3]
a == b    # True（值相等）
a is b    # False（不是同一个对象）
```

**Java 类比**：`is` ≈ `==`（引用比较），`==` ≈ `.equals()`（值比较）。这是最容易踩坑的地方。

---

## 5. 输入输出

```python
# 输出
print("hello")                    # hello
print("a", "b", sep=", ")        # a, b
print(f"name={name}, age={age}")  # f-string

# 输入
name = input("请输入名字: ")      # 返回 str
age = int(input("请输入年龄: "))   # 手动转类型
```

---

## 6. 类型转换

```python
int("42")       # 42
float("3.14")   # 3.14
str(42)         # "42"
bool(0)         # False
bool("")        # False
bool([])        # False
bool(1)         # True
bool("x")       # True
```

**注意**：Python 的 falsy 值比 Java 多：`0`, `0.0`, `""`, `[]`, `{}`, `()`, `set()`, `None` 都是 False。
