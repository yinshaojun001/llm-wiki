# 练习项目清单

> 每阶段一个练手项目，从简单到复杂

---

## 阶段一：基础语法（第 1-2 周）

### 项目：命令行计算器

**功能**：支持加减乘除、幂运算、括号优先级

```
> 2 + 3 * 4
14
> (2 + 3) * 4
20
> 2 ** 10
1024
```

**知识点**：输入输出、类型转换、条件判断、字符串处理

**扩展**：支持变量赋值（`x = 5`）和历史记录

---

## 阶段二：流程与函数（第 3-4 周）

### 项目：密码生成器

**功能**：
- 指定长度、包含的字符类型（大写/小写/数字/特殊字符）
- 生成多个密码
- 评估密码强度

```bash
python password_gen.py --length 16 --count 5 --no-special
```

**知识点**：函数定义、默认参数、`random` 模块、命令行参数

**扩展**：用 `argparse` 做正式的命令行参数解析

---

## 阶段三：数据结构（第 5-6 周）

### 项目：单词频率分析器

**功能**：
- 读取一个文本文件
- 统计每个单词出现的次数
- 输出 Top N 高频词
- 支持忽略停用词（the, a, is...）

**知识点**：文件读取、dict 操作、排序、推导式、`collections.Counter`

**扩展**：生成词云图（用 `wordcloud` 库）

---

## 阶段四：面向对象（第 7-8 周）

### 项目：图书馆管理系统

**功能**：
- `Book` 类：书名、作者、ISBN、状态（在馆/借出）
- `Library` 类：添加书籍、借书、还书、查询
- `Member` 类：会员信息、借阅历史
- 持久化到 JSON 文件

**知识点**：类设计、继承、魔术方法、`@dataclass`、文件 I/O

**扩展**：加一个 `@dataclass` 版本的 `Book`，对比代码量

---

## 阶段五：进阶精通（第 9-12 周）

### 项目 1（第 9-10 周）：异步网页爬虫

**功能**：
- 给定 URL 列表，并发抓取页面
- 提取标题、链接、图片
- 保存到 JSON 文件
- 支持限速和重试

**知识点**：`asyncio`、`httpx`、BeautifulSoup、装饰器（重试）

```python
# 核心架构
async def crawl(urls: list[str]) -> list[dict]:
    async with httpx.AsyncClient() as client:
        tasks = [fetch_page(client, url) for url in urls]
        return await asyncio.gather(*tasks)
```

### 项目 2（第 11-12 周）：REST API 服务

**功能**：
- 用 FastAPI 搭建 Todo API
- CRUD 接口
- Pydantic 数据校验
- SQLite 存储
- pytest 测试覆盖

```
GET    /todos          — 列表
POST   /todos          — 创建
GET    /todos/{id}     — 详情
PUT    /todos/{id}     — 更新
DELETE /todos/{id}     — 删除
```

**知识点**：FastAPI、Pydantic、SQLAlchemy、pytest、依赖注入

**扩展**：
- 加认证（JWT）
- 加分页
- 部署到 Fly.io / Railway
