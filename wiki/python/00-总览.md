# Python 从基础到精通

> 目标：系统掌握 Python，从基础语法到工程实战
> 对象：7 年 Java 后端工程师，零 Python 基础
> 时间：约 12 周（每周 6-8 小时）

---

## 为什么学 Python

- Agent 工程标配：LangChain / LlamaIndex / OpenAI SDK 全是 Python-first
- 数据处理 / 自动化脚本 / AI 原型，Java 太重，Python 刚好
- 你已有 OOP / 设计模式 / 工程化经验，学 Python 是降维，不是从零

## 学习原则

1. **类比优先**：每个新概念先找 Java 对应物，降低认知负担
2. **代码驱动**：看懂不等于会写，每节必须手敲示例
3. **项目检验**：每阶段结束必须完成一个练手项目
4. **不要背语法**：用 IDE 补全 + 查文档，把精力放在 Pythonic 思维上

---

## 前置要求

- 安装 Python 3.12+（推荐 pyenv 管理多版本）
- 安装 VS Code + Python 扩展
- 基本终端操作（你已具备）
