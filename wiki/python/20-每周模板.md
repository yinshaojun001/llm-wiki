# 每周学习模板

> 每周 6-8 小时，持续 12 周

---

## 周一：概念学习（2 小时）

- 阅读当周对应阶段的课程页面
- 手敲所有代码示例
- 记录疑问点

---

## 周三：动手练习（2 小时）

- 完成当周课程页面的练习题
- 遇到问题先查文档（docs.python.org），再搜 Stack Overflow
- 记录踩坑点

---

## 周五：项目实战（2-4 小时）

- 完成当周对应的练手项目（见 [[21-练习项目清单]]）
- 代码写在独立目录，提交到 Git
- 写简短的 README 说明

---

## 周末：回顾 + 扩展

- 回顾本周踩坑点，更新笔记
- 浏览 Python 标准库文档，发现新的有用的模块
- 如果有余力，尝试把 Java 项目中的一个小功能用 Python 重写

---

## 学习检查清单

每周结束时自问：

- [ ] 这周的核心概念我能用自己的话解释吗？
- [ ] 练习题都独立完成了吗？（不是抄答案）
- [ ] 练手项目能跑起来吗？
- [ ] 我知道 Java 里对应的做法是什么吗？
- [ ] 有没有什么地方让我觉得"Python 这样更简洁"？
