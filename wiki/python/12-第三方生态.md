> 最常用的第三方库，按场景分类

## 包管理

| 工具 | 说明 | 推荐度 |
|---|---|---|
| `pip` | 标准包管理器，最基础 | 入门用 |
| `uv` | Rust 写的超快包管理器，兼容 pip | **推荐** |
| `poetry` | 依赖锁定 + 打包发布 | 项目用 |
| `conda` | 数据科学专用，管理非 Python 依赖 | 数据科学用 |

```bash
# uv（推荐，比 pip 快 10-100 倍）
pip install uv
uv pip install requests
uv pip install -r requirements.txt

# poetry
pip install poetry
poetry init              # 初始化项目
poetry add requests      # 添加依赖
poetry install           # 安装所有依赖
poetry run python main.py
```

## HTTP 请求

### `requests`（最常用）

```python
import requests

# GET
r = requests.get("https://api.example.com/users", params={"page": 1})
r.status_code    # 200
r.json()         # 解析 JSON
r.text           # 原始文本

# POST
r = requests.post("https://api.example.com/users", json={"name": "Alice"})

# 带 Header
r = requests.get(url, headers={"Authorization": "Bearer xxx"})

# 超时
r = requests.get(url, timeout=5)

# Session（保持 Cookie）
s = requests.Session()
s.get("https://example.com/login")
s.get("https://example.com/dashboard")
```

### `httpx`（支持异步）

```python
import httpx

# 同步
r = httpx.get("https://api.example.com")

# 异步
async with httpx.AsyncClient() as client:
    r = await client.get("https://api.example.com")
```

**Java 类比**：`requests` ≈ `RestTemplate` / `HttpClient`，但更简洁。`httpx` ≈ `WebClient`。

## Web 框架

### `FastAPI`（推荐，现代异步框架）

```python
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()

class User(BaseModel):
    name: str
    age: int

@app.get("/users/{user_id}")
async def get_user(user_id: int):
    return {"user_id": user_id}

@app.post("/users")
async def create_user(user: User):
    return {"created": user.name}

# 自动生成 OpenAPI 文档：http://localhost:8000/docs
```

```bash
pip install fastapi uvicorn
uvicorn main:app --reload
```

### `Flask`（微框架，简单）

```python
from flask import Flask, jsonify

app = Flask(__name__)

@app.route("/users/<int:user_id>")
def get_user(user_id):
    return jsonify({"user_id": user_id})

if __name__ == "__main__":
    app.run(debug=True)
```

**Java 类比**：`FastAPI` ≈ Spring Boot（自动文档、参数校验）。`Flask` ≈ Javalin / Spark Java。

## 数据处理

### `pandas`（数据分析必备）

```python
import pandas as pd

# DataFrame（类似 Excel 表格）
df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie"],
    "age": [30, 25, 35],
    "city": ["NYC", "LA", "NYC"]
})

# 筛选
df[df["age"] > 25]              # 按条件筛选
df[["name", "age"]]             # 选列

# 聚合
df.groupby("city")["age"].mean()  # 按城市分组求平均

# 读写文件
df = pd.read_csv("data.csv")
df = pd.read_excel("data.xlsx")
df = pd.read_json("data.json")
df.to_csv("output.csv", index=False)
```

### `numpy`（数值计算）

```python
import numpy as np

a = np.array([1, 2, 3, 4, 5])
a * 2              # array([2, 4, 6, 8, 10])  向量化运算
np.mean(a)         # 3.0
np.dot(a, a)       # 55  点积
a.reshape(5, 1)    # 变形
```

**Java 类比**：`pandas` ≈ 没有直接对应（Java 的表格处理很弱）。`numpy` ≈ Apache Commons Math。

## 数据验证

### `pydantic`（数据校验 + 序列化）

```python
from pydantic import BaseModel, EmailStr, Field

class User(BaseModel):
    name: str = Field(..., min_length=1, max_length=50)
    age: int = Field(..., ge=0, le=150)
    email: EmailStr
    tags: list[str] = []

# 自动校验
user = User(name="Alice", age=30, email="alice@example.com")
user.model_dump()          # 转 dict
user.model_dump_json()     # 转 JSON 字符串

# 校验失败会抛 ValidationError
User(name="", age=-1, email="bad")  # ValidationError!
```

**Java 类比**：`pydantic` ≈ Bean Validation（`@Valid`）+ Jackson，但更强大。FastAPI 用 pydantic 做参数校验。

```python

import httpx

@respx.mock
def test_api():
    respx.get("https://api.example.com/users").mock(return_value=httpx.Response(200, json=[]))
    r = httpx.get("https://api.example.com/users")
    assert r.json() == []
```

## CLI 工具

### `click`（命令行参数解析）

```python
import click

@click.command()
@click.option("--name", prompt="Your name", help="Name to greet")
@click.option("--count", default=1, help="Number of greetings")
def hello(name, count):
    for _ in range(count):
        click.echo(f"Hello, {name}!")

if __name__ == "__main__":
    hello()
```

## 速查：选哪个

| 场景 | 推荐 | 备选 |
|---|---|---|
| HTTP 请求 | `requests` | `httpx`（需异步时） |
| Web API | `FastAPI` | `Flask` |
| 数据分析 | `pandas` | `polars`（更快） |
| 数据验证 | `pydantic` | `dataclasses`（简单场景） |
| 测试 | `pytest` | `unittest`（标准库） |
| CLI | `click` | `argparse`（标准库） |
| 环境变量 | `python-dotenv` | `os.environ` |
| 日志 | `loguru` | `logging`（标准库） |
| 终端美化 | `rich` | — |
| 包管理 | `uv` | `pip` / `poetry` |
