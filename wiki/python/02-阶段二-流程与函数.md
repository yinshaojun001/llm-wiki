# 阶段二：流程控制与函数（第 3-4 周）

> 目标：掌握分支、循环、函数定义，能写有逻辑的程序

---

## 1. 条件判断

```python
score = 85

# if-elif-else（没有 switch，没有大括号）
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 60:
    grade = "C"
else:
    grade = "D"

# 三元表达式
grade = "A" if score >= 90 else "B"

# match-case（Python 3.10+，类似 Java switch）
match command:
    case "start":
        start()
    case "stop":
        stop()
    case _:
        print("unknown")
```

**注意**：Python 用缩进（4 空格）代替大括号。缩进错误 = 语法错误。不要混用 Tab 和空格。

## 2. 循环

```python
# for 循环（遍历可迭代对象）
for i in range(5):          # 0,1,2,3,4
    print(i)

for i in range(2, 10, 3):   # 2,5,8（start, stop, step）
    print(i)

names = ["Alice", "Bob", "Charlie"]
for name in names:
    print(name)

# 带索引遍历
for i, name in enumerate(names):
    print(f"{i}: {name}")

# while 循环
count = 0
while count < 5:
    print(count)
    count += 1

# break / continue（和 Java 一样）
for i in range(10):
    if i == 3:
        continue  # 跳过 3
    if i == 7:
        break     # 到 7 停止
    print(i)
```

**Java 类比**：`for name in names` ≈ Java 的增强 for 循环。`enumerate()` 没有直接对应，需要手动维护索引。

## 3. 函数

```python
# 基本定义
def greet(name: str) -> str:
    return f"Hello, {name}"

# 默认参数
def greet(name: str, greeting: str = "Hello") -> str:
    return f"{greeting}, {name}"

greet("Alice")              # "Hello, Alice"
greet("Alice", "Hi")       # "Hi, Alice"

# 关键字参数（调用时指定参数名）
greet(greeting="Hey", name="Alice")  # 顺序可以变

# 可变参数
def calc_sum(*args):        # args 是 tuple
    return sum(args)

calc_sum(1, 2, 3, 4)       # 10

# 关键字可变参数
def print_info(**kwargs):   # kwargs 是 dict
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=30)

# 多返回值（实际返回 tuple）
def get_user():
    return "Alice", 30      # 等价于 return ("Alice", 30)

name, age = get_user()      # 解包
```

**Java 类比**：
- `*args` ≈ 可变参数 `String... args`
- `**kwargs` ≈ 没有直接对应，类似 `Map<String, Object>`
- 多返回值 Java 需要包装类或 record

## 4. 作用域

```python
x = 10          # 全局

def foo():
    x = 20      # 局部（遮蔽全局）
    print(x)    # 20

foo()
print(x)        # 10（全局不受影响）

# 如果要修改全局变量
def bar():
    global x
    x = 30

bar()
print(x)        # 30
```

**最佳实践**：尽量避免 `global`，用参数和返回值传递数据。

## 5. Lambda

```python
# 匿名函数（一行表达式）
square = lambda x: x ** 2
square(5)   # 25

# 常用于排序
users = [("Alice", 30), ("Bob", 25)]
users.sort(key=lambda u: u[1])   # 按年龄排序

# map / filter（函数式风格）
nums = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x**2, nums))       # [1, 4, 9, 16, 25]
evens = list(filter(lambda x: x % 2 == 0, nums)) # [2, 4]
```

**Java 类比**：lambda ≈ Java 的 lambda。但 Python 的 lambda 只能是单行表达式，复杂逻辑用 `def`。

## 6. 模块与包

```python
# 导入模块
import math
math.sqrt(16)  # 4.0

# 导入特定函数
from math import sqrt, pi
sqrt(16)       # 4.0

# 别名
import numpy as np

# 自己的模块
# 文件: utils.py
def say_hello(name):
    return f"Hello, {name}"

# 文件: main.py
from utils import say_hello
say_hello("Alice")
```

**包结构**：
```
mypackage/
├── __init__.py      # 标识这是一个包（可以为空）
├── core.py
└── utils.py
```

**Java 类比**：Python 的 `import` ≈ Java 的 `import`。`__init__.py` ≈ `package-info.java`。但 Python 没有 `public/private` 访问控制。
