# Python 常用标准库速查

> 最常用的标准库模块，按场景分类

## 文件与路径

### `pathlib`（推荐，面向对象）

```python
from pathlib import Path

p = Path("data") / "users" / "info.json"  # 路径拼接
p.exists()           # 是否存在
p.is_file()          # 是否是文件
p.is_dir()           # 是否是目录
p.read_text()        # 读取文本
p.read_bytes()       # 读取二进制
p.write_text("hi")   # 写入文本
p.suffix             # ".json"
p.stem               # "info"
p.parent             # Path("data/users")
p.glob("*.txt")      # 匹配文件
p.rglob("*.txt")     # 递归匹配
```

### `os`（底层操作）

```python
import os

os.getcwd()                  # 当前目录
os.listdir(".")              # 列出目录
os.makedirs("a/b/c", exist_ok=True)  # 创建目录
os.environ["HOME"]           # 环境变量
os.path.join("a", "b")      # 路径拼接（旧方式，用 pathlib 替代）
```

**Java 类比**：`pathlib.Path` ≈ `java.nio.file.Path`，比 `os.path` 更 Pythonic。

## JSON

```python
import json

# 序列化
data = {"name": "Alice", "age": 30}
s = json.dumps(data, ensure_ascii=False, indent=2)
json.dump(data, open("out.json", "w"), ensure_ascii=False)

# 反序列化
obj = json.loads('{"name": "Alice"}')
obj = json.load(open("out.json"))
```

**Java 类比**：`json` ≈ Jackson，但内置在标准库中，无需引入依赖。

## 正则表达式

```python
import re

# 基本匹配
re.search(r"\d+", "abc123def")     # 匹配第一个数字
re.findall(r"\d+", "a1b2c3")       # ['1', '2', '3']
re.sub(r"\d+", "*", "a1b2c3")      # "a*b*c*"

# 分组
m = re.search(r"(\w+)@(\w+)\.(\w+)", "alice@example.com")
m.group(0)   # "alice@example.com"（整个匹配）
m.group(1)   # "alice"
m.group(2)   # "example"

# 编译（多次使用时更快）
email_pattern = re.compile(r"^[\w.+-]+@[\w-]+\.[\w.]+$")
email_pattern.match("alice@example.com")
```

**Java 类比**：`re` ≈ `java.util.regex`，但 API 更简洁。

## 日期时间

```python
from datetime import datetime, timedelta

now = datetime.now()                     # 当前时间
now.strftime("%Y-%m-%d %H:%M:%S")       # 格式化
datetime.strptime("2026-05-07", "%Y-%m-%d")  # 解析
now + timedelta(days=7)                  # 7 天后
(now - datetime(2026, 1, 1)).days        # 天数差

# 更好用的第三方库
# pip install arrow / pendulum
```

**Java 类比**：`datetime` ≈ `java.time.LocalDateTime`，但功能更弱。复杂时间处理推荐 `pendulum`。

## 集合扩展

### `collections`

```python
from collections import Counter, defaultdict, deque, namedtuple

# Counter（计数器）
words = ["a", "b", "a", "c", "a"]
Counter(words)             # Counter({"a": 3, "b": 1, "c": 1})
Counter(words).most_common(2)  # [("a", 3), ("b", 1)]

# defaultdict（带默认值的 dict）
d = defaultdict(list)
d["users"].append("Alice")  # 不需要先初始化

# deque（双端队列，两端 O(1) 插入/删除）
q = deque([1, 2, 3])
q.appendleft(0)    # [0, 1, 2, 3]
q.pop()            # 3
q.popleft()        # 0
```

**Java 类比**：`Counter` ≈ 没有直接对应（需要 `stream().collect(groupingBy(..., counting()))`）。`defaultdict` ≈ `computeIfAbsent`。`deque` ≈ `ArrayDeque`。

## `itertools`

```python
import itertools

# 组合
list(itertools.chain([1, 2], [3, 4]))       # [1, 2, 3, 4]
list(itertools.product([1, 2], ["a", "b"])) # [(1,'a'), (1,'b'), (2,'a'), (2,'b')]
list(itertools.combinations([1,2,3], 2))    # [(1,2), (1,3), (2,3)]
list(itertools.permutations([1,2,3], 2))    # [(1,2), (1,3), (2,1), ...]

# 分组
data = [("a", 1), ("a", 2), ("b", 3)]
for key, group in itertools.groupby(data, key=lambda x: x[0]):
    print(key, list(group))

# 无限迭代器
counter = itertools.count(1)     # 1, 2, 3, ...
cycle = itertools.cycle([0, 1])  # 0, 1, 0, 1, ...
```

## `functools`

```python
import functools

# 缓存（记忆化）
@functools.lru_cache(maxsize=128)
def fibonacci(n):
    if n < 2:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# 偏函数
int2 = functools.partial(int, base=2)
int2("1010")  # 10

# 归约
functools.reduce(lambda a, b: a + b, [1, 2, 3, 4])  # 10
```

## `logging`

```python
import logging

logging.basicConfig(level=logging.INFO, format="%(asctime)s %(levelname)s %(message)s")
logger = logging.getLogger(__name__)

logger.info("Starting...")
logger.warning("Low memory")
logger.error("Failed", exc_info=True)
```

**Java 类比**：`logging` ≈ SLF4J + Logback。但 Python 标准库的 logging 配置较繁琐，很多人用 `loguru` 替代。

## `subprocess`

```python
import subprocess

# 简单执行
result = subprocess.run(["ls", "-la"], capture_output=True, text=True)
print(result.stdout)
print(result.returncode)

# 管道
result = subprocess.run("echo hello | tr a-z A-Z", shell=True, capture_output=True, text=True)
print(result.stdout)  # "HELLO\n"
```

**Java 类比**：`subprocess.run()` ≈ `ProcessBuilder`，但更简洁。
