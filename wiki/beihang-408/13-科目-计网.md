## 408考纲全量

### 第一章 概述
- OSI七层 vs TCP/IP四层
- 性能指标：带宽(bps)、时延(发送+传播+处理+排队)、时延带宽积、RTT、吞吐量

### 第二章 物理层
- 奈奎斯特(无噪声)：最大速率 = 2W log₂V
- 香农(有噪声)：最大速率 = W log₂(1+S/N)，SNR=10log₁₀(S/N) dB
- 编码：NRZ/曼彻斯特(以太网)/差分曼彻斯特
- 交换：电路交换(电话) vs 报文交换 vs 分组交换(数据报/虚电路)

### 第三章 数据链路层

**差错检测**：CRC多项式除法求余数（手算）

**CSMA/CD**：
- 先听后发、边发边听、冲突停发、随机重发
- 最小帧长 = 2×传播时延×带宽
- 二进制指数退避：第k次重试，等待[0, 2^k-1]个争用期

**交换机**：自学习（记源MAC→端口）、转发（目的MAC已知→指定端口，未知→广播）

### 第四章 网络层

**IP子网划分（每年必考）**：
```
给定: 192.168.1.0/24，划分4个子网
→ 借2位主机位: /26
→ 子网1: 192.168.1.0/26，可用 1-62，广播 63
→ 子网2: 192.168.1.64/26，可用 65-126，广播 127
→ 子网3: 192.168.1.128/26，可用 129-190，广播 191
→ 子网4: 192.168.1.192/26，可用 193-254，广播 255
```

**路由聚合（CIDR）**：多个子网取最长公共前缀

**路由协议**：
| | RIP | OSPF | BGP |
|---|---|---|---|
| 算法 | 距离向量 | 链路状态 | 路径向量 |
| 协议 | UDP/520 | IP/89 | TCP/179 |
| 范围 | 域内(AS内) | 域内 | 域间(AS间) |
| 度量 | 跳数 | 代价(带宽) | 策略 |

### 第五章 传输层（重点）

**TCP三次握手**：
```
Client                    Server
SYN=1, seq=x       →
                   ←  SYN=1, ACK=1, seq=y, ack=x+1
ACK=1, seq=x+1, ack=y+1 →
```

**TCP四次挥手**：
```
Client                    Server
FIN=1, seq=u       →
                   ←  ACK=1, seq=v, ack=u+1
                   ←  FIN=1, ACK=1, seq=w, ack=u+1
ACK=1, seq=u+1, ack=w+1 →
(TIME_WAIT 2MSL)
```

**TCP拥塞控制**：
- 慢开始：每轮cwnd×2，直到ssthresh
- 拥塞避免：每轮cwnd+1
- 快重传：收到3个重复ACK→立刻重传
- 快恢复：ssthresh=cwnd/2，cwnd=ssthresh，进入拥塞避免

**TCP vs UDP**：
| | TCP | UDP |
|---|---|---|
| 连接 | 面向连接 | 无连接 |
| 可靠 | 确认+重传 | 不保证 |
| 首部 | 20字节 | 8字节 |
| 流量控制 | 滑动窗口 | 无 |
| 拥塞控制 | 有 | 无 |
| 适用 | HTTP/FTP/SMTP | DNS/DHCP/视频流 |

### 第六章 应用层
- DNS：UDP/53，递归查询 vs 迭代查询
- FTP：控制连接TCP/21 + 数据连接TCP/20（主动/被动模式）
- SMTP/POP3：发TCP/25，收TCP/110
- HTTP：TCP/80，HTTP/1.1持久连接，GET/POST/PUT/DELETE
- 状态码：200 OK / 301永久重定向 / 302临时重定向 / 403禁止 / 404未找到 / 500服务器错误
