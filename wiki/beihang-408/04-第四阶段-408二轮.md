# 第四阶段：408二轮融合 + 真题

> 2027年5月-8月（17周）| 从"学会"到"会考"

一轮你分别学了四门课。二轮做三件事：打通交叉点、上真题、补一轮漏掉的。

---

## 五月（4周）：DS二轮 + 算法题专项

### 第1-2周
- [ ] 王道DS二轮速过：一章一章来，只看一轮标注的错题和模糊点
- [ ] **每天一道算法题**，限时30分钟手写C代码：
  - 周一：链表题（反转/合并/找环）
  - 周二：二叉树题（遍历/高度/宽度/祖先）
  - 周三：查找题（二分变体/哈希）
  - 周四：排序题（快排变体/归并应用/堆应用）
  - 周五：图题（BFS/DFS 应用）
- [ ] 做：王道DS强化课大题部分

### 第3-4周
- [ ] DS代码题第二轮：闭卷写，15分钟一道
- [ ] 时间复杂度秒出（给代码段，立刻算出O）
- [ ] 做：王道课后真题大题 第二章2.10、2.11、2.12、2.13（全是408真题）
- [ ] **检验**：随便挑一道408数据结构真题大题，20分钟内写出正确代码

**LeetCode辅助**：搜索"408考研题单"，挑链表/树/排序的Easy-Medium题练手感。

---

## 六月（4周）：计组二轮 ← 本轮重点

### 第5-6周
- [ ] 王道计组二轮速过，重点清三个板块：
  1. **浮点数IEEE754**：单精度/双精度转换，浮点加减模拟
  2. **Cache**：三种映射地址划分、LRU替换、命中率计算
  3. **数据通路+流水线**：单周期CPU、五级流水、冒险处理
- [ ] 做：2016-2024年408真题中计组部分，专练

### 第7-8周
- [ ] 计组大题专项，每类练满10道：
  - Cache映射+命中率计算
  - 数据通路（给指令，写微操作序列）
  - 流水线（流水时空图、吞吐率加速比）
  - 芯片扩展（地址线/数据线/片选线计算）
- [ ] 做：王道计组强化课全部大题
- [ ] **检验**：408计组真题大题，连续3道正确

**交叉点注意**：虚拟内存（OS第三章）和Cache（计组第三章）对照学，本质都是"缓存+地址映射"。

---

## 七月（4周）：OS + 计网 二轮

### 第9-10周 OS二轮
- [ ] **PV操作三题**闭卷默写：生产者-消费者 / 读者-写者 / 哲学家
- [ ] 内存管理大题：页表地址转换（逻辑→物理）、页面置换（OPT/LRU/FIFO/Clock）手写模拟
- [ ] 文件物理结构大题：混合索引分配，多级索引能寻址的最大文件大小
- [ ] 做：408真题OS部分（2016-2024），专项练

### 第11周 计网二轮
- [ ] IP子网划分专项：10道题练到15分钟全对
- [ ] TCP综合大题：三次握手/四次挥手状态图 + 拥塞控制cwnd曲线
- [ ] CSMA/CD：最小帧长计算
- [ ] 做：408真题计网部分（2016-2024），专项练

### 第12周 OS+计组交叉专题
- [ ] 虚拟内存页表 ↔ TLB ↔ Cache ↔ 主存，四级地址转换全链路
- [ ] 能画出从逻辑地址→物理地址的完整流程：TLB命中/不命中→页表→Cache→主存
- [ ] 做：408真题中同时考OS+计组的综合题

## 八月（5周）：408真题第一轮

### 第13-17周
- [ ] **2016-2024年408真题，共9套，每周2套**
- [ ] 每套严格限时3小时（和真实考试一样）
- [ ] 做完后精析：
  - 选择题：每个选项为什么对、为什么错（不是只记答案）
  - 大题：标准答案的答题规范（计组数据通路要画图、OS PV操作要有注释）
- [ ] 错题归类到三类：知识点遗漏 / 审题粗心 / 时间不够
- [ ] 二刷错题

**目标**：选择题正确率>70%（>28/40，即>56/80分），总分>90/150

## 数学二同步

> 目标：真题一刷完成 → 真题二刷 → 模拟题
> 从"会做"到"能在规定时间做对"
> 每章对照 [[14-科目-数学二]] 查漏补缺

### 五月（4周）：真题一刷收尾 + 错题专项

#### 第1-2周：真题一刷收尾（2013-2024）
- [ ] 如果第三阶段真题没做完，这两周做完
- [ ] 每套做完精析，不追求数量，追求每道错题搞懂
- [ ] 错题按章节归类：极限/导数/中值定理/不定积分/定积分/定积分应用/微分方程/多元微分/二重积分/行列式/矩阵/向量组/方程组/特征值/二次型

#### 第3-4周：错题专项突破
- [ ] 统计错题分布，找出最薄弱的三章
- [ ] 每个薄弱章节：回顾[[14-科目-数学二]]概念 → 重做章节基础题 → 重做错题
- [ ] **计算错误专项**：每天10道纯计算题（不定积分/定积分/二重积分），目标是连续10道计算不出错
- [ ] **检验**：最薄弱章节的同类题，正确率从<50%提升到>70%

### 六月（4周）：真题二刷

#### 第5-8周：真题二刷（2013-2024，12套，每周2-3套）
- [ ] 只做一刷时做错的题 + 一刷时感觉困难的题
- [ ] 不用做整套，重点攻克错题
- [ ] 二刷还错的题 → 标红色，这是考前最后看的
- [ ] **二刷不是重新做一遍，是验证一刷的错误已经改正**
- [ ] 到6月底目标：
  - [ ] 一刷错题二刷后正确率>90%
  - [ ] 计算题不出错（不定积分/定积分/二重积分）
  - [ ] 线代大题步骤规范（方程组→行阶梯→自由变量→通解；二次型→特征值→特征向量→正交化→标准型）

### 七月（4周）：模拟题 + 查漏补缺

#### 第9-12周：模拟题阶段
- [ ] **张宇8套卷 或 李林6套卷**，每周1-2套，限时3小时
- [ ] 模拟题比真题难，分数低不要慌。模拟题的意义是暴露盲区
- [ ] 每套做完精析，错题补充到错题本
- [ ] 模拟题中的新型题（真题没见过的考法）单独标注
- [ ] 同时维护计算手感：每天5道纯计算（不定积分/定积分/二重积分/一阶线性微分方程）
- [ ] **到7月底目标**：模拟题均分>90/150

### 八月（5周）：真题三刷选择 + 回归基础

#### 第13-15周：真题选择+填空专项
- [ ] 2013-2024真题所有选择+填空重做（12套×约10道选填=120道）
- [ ] 选填是拿分关键（56分/150分），目标是选填正确率>85%（>48分）
- [ ] 常见选填陷阱整理（等价无穷小替换条件、洛必达使用条件、极值判别AC-B²、正定性判定...）

#### 第16-17周：回归基础 + 公式默写
- [ ] [[14-科目-数学二]]公式卡片全部默写一遍
- [ ] 导数表、积分表、微分方程通解公式——必须闭卷默写全对
- [ ] 错题本回顾：红标错题（二刷还错的）再做一遍
- [ ] **第四阶段数学检验**：
  - [ ] 真题二刷完成，正确率>90%
  - [ ] 选填正确率>85%
  - [ ] 公式卡片闭卷默写全对
  - [ ] 模拟题均分>90
