## 408考纲全量

### 第一章 概述
- OS四个特征：并发、共享、虚拟、异步
- 内核态 vs 用户态（CPU模式位）
- 中断 vs 异常 vs 系统调用
- 宏内核 vs 微内核 vs 外核

### 第二章 进程管理

**进程 vs 线程**：
| | 进程 | 线程 |
|---|---|---|
| 资源拥有 | 独立地址空间 | 共享进程地址空间 |
| 调度 | 进程切换开销大 | 线程切换开销小 |
| 通信 | IPC（管道/消息队列/共享内存） | 直接读写共享变量 |

**CPU调度算法**：FCFS / SJF / 优先级 / 高响应比优先 / RR / 多级队列 / 多级反馈队列

**同步与互斥**：
- 临界区四个原则：互斥、有空让进、有限等待、让权等待
- 信号量PV操作模板：
  ```
  semaphore mutex = 1;  // 互斥信号量
  semaphore empty = N;  // 空闲缓冲区数
  semaphore full = 0;   // 已用缓冲区数
  
  Producer:                    Consumer:
    produce item;                P(full);
    P(empty);                    P(mutex);
    P(mutex);                    取数据;
    放数据;                      V(mutex);
    V(mutex);                    V(empty);
    V(full);                     consume item;
  ```
- **三个经典问题手写**：生产者-消费者 / 读者-写者 / 哲学家进餐

**死锁**：
- 四个必要条件：互斥、不剥夺、请求保持、循环等待
- 银行家算法：Max矩阵 / Allocation矩阵 / Need矩阵 / Available向量

### 第三章 内存管理

**分页地址转换**：
- 逻辑地址 = 页号 | 页内偏移
- 物理地址 = 页框号×页大小 + 页内偏移
- TLB：页表的Cache，快但小

**页面置换算法**：
| 算法 | 规则 | 特点 |
|---|---|---|
| OPT | 淘汰将来最远使用的 | 理论最优，无法实现 |
| FIFO | 淘汰最早进入的 | Belady异常 |
| LRU | 淘汰最近最少用的 | 最好但实现复杂 |
| Clock | 环形扫描，访问位=0淘汰 | 实际最常用 |
| 改进Clock | 同时看访问位和修改位 | 减少写回开销 |

### 第四章 文件管理

**文件物理结构**：
- 连续分配：快但易碎片
- 链接分配：隐式(每个块存下一块指针)/显式(FAT表)
- 索引分配：一个索引块存所有数据块指针
- **混合索引**：10个直接块 + 1个一级间接 + 1个二级间接 + 1个三级间接
  - 例：块大小4KB，指针占4B，每块可存1024个指针
  - 直接寻址：10×4KB = 40KB
  - 一级间接：1024×4KB = 4MB
  - 二级间接：1024²×4KB = 4GB
  - 三级间接：1024³×4KB = 4TB

**inode**：含文件属性(权限/所有者/大小/时间) + 数据块指针

### 第五章 I/O管理
- 磁盘调度：FCFS<SSTF<SCAN<C-SCAN<LOOK
- SPOOLing：独占设备变共享设备
- 单缓冲处理时间 = max(C,T)+M，双缓冲 = max(C+M, T)
