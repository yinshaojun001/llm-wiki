### 十月（4周）：二重积分二轮 + 线代启动

#### 第5周（10.1-10.10，含国庆）二重积分 二轮
- [ ] 汤家凤强化课：二重积分
- [ ] 交换积分次序综合题（先换序再计算，含参数的区域）
- [ ] 极坐标和直角坐标的选择判断力
- [ ] 利用对称性简化计算（奇函数在对称区域积分为0）
- [ ] 含绝对值的二重积分（分区域去绝对值）
- [ ] 做：1800题二重积分强化篇

#### 第6周（10.12-10.17）线代启动：行列式
- [ ] 汤家凤线代基础课：行列式
- [ ] **行列式的本质是一个数**，|A|≠0↔可逆
- [ ] 性质是算行列式的捷径：行变换不影响值（除换行变号）、上三角=对角线乘积
- [ ] 4阶+行列式降阶算法（消成上三角）
- [ ] 范德蒙行列式、爪型行列式
- [ ] 做：线代讲义行列式章节基础题

#### 第7周（10.19-10.24）矩阵
- [ ] 汤家凤线代基础课：矩阵运算
- [ ] 矩阵乘法不满足交换律（AB≠BA），这是新手最容易掉的坑
- [ ] 逆矩阵：2阶求逆公式 `[[a,b],[c,d]]^(-1) = 1/(ad-bc)·[[d,-b],[-c,a]]`
- [ ] 矩阵的秩 r(A)：非零子式的最高阶数 = 行阶梯形中非零行数
- [ ] 矩阵方程：AX=B→X=A^(-1)B, XA=B→X=BA^(-1)
- [ ] 分块矩阵（分块对角阵的逆=各块分别求逆）
- [ ] 做：矩阵章节基础题

#### 第8周（10.26-10.31）向量组
- [ ] 汤家凤线代基础课：向量组
- [ ] 线性相关 vs 线性无关（有冗余 vs 都独立）
- [ ] 判别法：向量个数>维数→必相关；r(A)<向量个数→相关
- [ ] 极大无关组 = 向量组的"骨架"（挑出几个独立向量，其余都可由它们表出）
- [ ] 线性表出：β能由α们表出 → 解方程组 → 有解就是能
- [ ] 做：向量组章节基础题
- [ ] **月末检验**：线代前三章的概念不混（秩、相关/无关、表出之间的关系）

### 十一月（4周）：线代 + 二重积分保持手感

#### 第9周（11.2-11.7）线性方程组
- [ ] 汤家凤线代基础课：线性方程组
- [ ] **解的存在唯一性判定**（最基础也最重要）：
  - Ax=b有解 ⇔ r([A|b])=r(A)
  - 唯一解：有解且r(A)=n；无穷解：有解且r(A)<n
  - Ax=0有非零解 ⇔ r(A)<n
- [ ] 求齐次通解：行阶梯→确定自由变量→基础解系
- [ ] 求非齐次通解：特解+齐次通解
- [ ] 含参数的方程组讨论（参数不同值r(A)可能不同）
- [ ] 做：方程组章节基础题，含参数讨论题练5道

#### 第10周（11.9-11.14）特征值与特征向量
- [ ] 汤家凤线代基础课：特征值与特征向量
- [ ] **核心定义**：Ax=λx，A作用在x上只伸缩、不转向
- [ ] 求特征值：|A-λE|=0→特征方程→解λ
- [ ] 求特征向量：对每个λ，解(A-λE)x=0
- [ ] 性质：tr(A)=Σλᵢ（迹=特征值之和），|A|=Πλᵢ（行列式=特征值之积）
- [ ] 相似对角化：P⁻¹AP=Λ，P由特征向量按列排成
- [ ] 做：特征值章节基础题

#### 第11周（11.16-11.21）二次型
- [ ] 汤家凤线代基础课：二次型
- [ ] 二次型矩阵A是对称矩阵，f=x^T A x
- [ ] **正交变换化标准型**（大题核心）：
  ① 写A ② 求特征值 ③ 求特征向量 ④ 正交化单位化 ⑤ x=Qy得标准型
- [ ] 正定性判定：特征值全部>0 / 顺序主子式全部>0
- [ ] 配方法化标准型（另一种方法，也要会）
- [ ] 做：二次型章节基础题

- [ ] 线代章节关系梳理（行列式→矩阵→向量→方程组→特征值→二次型，一环扣一环）
- [ ] 做：线代全书基础题回顾 + 错题整理
