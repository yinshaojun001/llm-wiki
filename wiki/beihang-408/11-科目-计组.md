# 计算机组成原理（45分）

> 选择约11题(22分) + 大题2题(23分)
> 408最难的科目。考的是"软硬件协同"思维，不是背概念。

---

## 408考纲全量

### 第一章 计算机系统概述
- 冯诺依曼结构：运算器+控制器+存储器+输入+输出，存储程序
- 性能指标：
  - **CPU执行时间** = 指令条数 × CPI / 主频
  - MIPS = 指令条数/(执行时间×10^6)
  - MFLOPS = 浮点操作数/(执行时间×10^6)

### 第二章 数据的表示和运算

**定点数**：
- 原码/反码/补码/移码 → 补码是王道，计算全用它
- 补码加减：`[A+B]补 = [A]补+[B]补`
- 溢出判断：双符号位（00=正，11=负，01=上溢，10=下溢）

**浮点数IEEE754（每年必考）**：
- 单精度 float：1位符号 + 8位阶码(偏置127) + 23位尾数
- 双精度 double：1 + 11(偏置1023) + 52
- 真值 = (-1)^s × 1.M × 2^(E-127)
- 特殊值：E全0 M全0=0，E全1 M全0=∞，E全1 M非0=NaN

### 第三章 存储器层次结构

**主存与CPU连接**：
- 芯片数 = (总容量/单片容量) × (总位宽/单片位宽)
- 地址线数 = log₂(字数)，数据线数 = 位宽

**Cache（大题核心）**：
- 地址划分：`主存地址 = 标记 | 组号 | 块内偏移`
- 直接映射：组数 = Cache行数
- 全相联：组数 = 1
- L路组相联：组数 = Cache行数 / L
- 替换算法：LRU（计组考最多）、FIFO、随机
- 写策略：写直达(同时写Cache和主存) vs 写回(只写Cache，替换时写回)
- 写分配 vs 非写分配

**虚拟存储器**：页表、TLB、地址转换 → 和[[12-科目-OS]]联动

### 第四章 指令系统
- 九种寻址方式：立即/直接/间接/寄存器/寄存器间接/基址/变址/相对/堆栈
- CISC(复杂指令集) vs RISC(精简指令集)
- 大端(高字节低地址) vs 小端(低字节低地址)
- 函数调用栈帧：`push ebp; mov ebp, esp; sub esp, N`

### 第五章 中央处理器

**数据通路（大题核心）**：
- 单周期CPU：所有指令在一个时钟周期完成
- 指令执行五阶段：取指IF→译码ID→执行EX→访存MEM→写回WB

**流水线（大题核心）**：
- 五级流水：IF/ID/EX/MEM/WB
- 吞吐率 TP = n / ((k + n - 1) × Δt)
- 实际加速比 Sp = T顺序/T流水
- 三种冒险：
  - 数据冒险：用转发(Forwarding)解决，Load-use冒险需加气泡
  - 控制冒险：分支预测、延迟槽
  - 结构冒险：增加硬件资源

### 第六章 总线和输入输出系统
- 三种I/O方式对比：

| | 程序查询 | 中断 | DMA |
|---|---|---|---|
| 数据传送 | CPU | CPU | DMA控制器 |
| CPU干预 | 全程 | 每次数据 | 仅开始+结束 |
| 速度 | 慢 | 中 | 快 |
| 适用 | 低速外设 | 中速 | 高速块设备 |
