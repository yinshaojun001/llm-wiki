- [ ] 无穷小比较：等价无穷小替换是后面所有极限题的基础
- [ ] 间断点分类：第一类（可去/跳跃）vs 第二类（无穷/振荡）
- [ ] 做：汤家凤讲义第一章基础题
- [ ] **检验**：给一道0/0型极限，能用等价无穷小替换做出来

#### 第2周（5.11-5.17）极限计算强化
- [ ] 汤家凤基础课：极限运算法则
- [ ] **重点练五种极限类型**：0/0型、∞/∞型、1^∞型、∞-∞型、0·∞型
- [ ] 洛必达法则的使用条件（0/0或∞/∞才能用）
- [ ] 做：汤家凤讲义极限章节剩余题 + 1800题极限基础篇
- [ ] **检验**：每种类型的极限题各练5道，2分钟内做完一道

#### 第3周（5.18-5.24）连续性与导数定义
- [ ] 汤家凤基础课：函数连续性、导数定义
- [ ] 连续的三条件：①f(x₀)有定义 ②极限存在 ③极限=f(x₀)
- [ ] **导数定义**：`f'(x₀)=lim(h→0)[f(x₀+h)-f(x₀)]/h` —— 可导必连续，连续不一定可导
- [ ] 做：连续与导数章节基础题
- [ ] **检验**：用导数定义求 f(x)=|x| 在 x=0 处的左右导数

### 六月（4周）：中值定理 + 积分

#### 第5周（6.1-6.7）微分中值定理
- [ ] 汤家凤基础课：中值定理
- [ ] **重点**：罗尔定理（f(a)=f(b)→存在ξ使f'(ξ)=0）、拉格朗日（平均值=某点导数）
- [ ] 罗尔定理解题关键：**构造原函数**
- [ ] 柯西、泰勒第一遍可以先了解，不强求
- [ ] 做：中值定理基础题（只做罗尔+拉格朗日，各5-8道）
- [ ] **检验**：会构造简单原函数（如f'(ξ)+f(ξ)=0→构造eˣf(x)）

#### 第6周（6.8-6.14）不定积分 —— 最重要的基本功
- [ ] 汤家凤基础课：不定积分概念 + 第一类换元法（凑微分）
- [ ] **这是整个微积分的核心**：积不出来，后面定积分、微分方程全卡死
- [ ] 凑微分常见模式（对照[[14-科目-数学二]]）：∫f(ax+b)dx、∫f(eˣ)eˣdx、∫f(lnx)/x dx
- [ ] 每天练10道凑微分题，形成条件反射
- [ ] 做：不定积分基础题（凑微分部分）

#### 第7周（6.15-6.21）不定积分续
- [ ] 汤家凤基础课：第二类换元法 + 分部积分法
- [ ] 第二类换元（三角代换）：√(a²-x²)→x=asint, √(a²+x²)→x=atant, √(x²-a²)→x=asect
- [ ] **分部积分**：∫udv = uv - ∫vdu，"反对幂指三"口诀选u
- [ ] 三种经典分部积分：∫xeˣdx、∫xsinxdx、∫lnx dx —— 必须闭卷写出
- [ ] 做：不定积分剩余基础题

#### 第8周（6.22-6.28）定积分
- [ ] 汤家凤基础课：定积分概念 + 牛顿-莱布尼茨公式
- [ ] **变上限函数求导**：Φ(x)=∫[a,x]f(t)dt → Φ'(x)=f(x)；上限是函数则链式法则
- [ ] 定积分换元：**换元必须换限！**
- [ ] 对称区间奇偶性：奇函数→0，偶函数→2倍[0,a]
- [ ] 做：定积分基础题
- [ ] **月末检验**：随机10道不定积分，8道以上一次做对

### 七月（4周）：定积分应用 + 微分方程

#### 第9周（6.29-7.5）定积分应用（一）

#### 第10周（7.6-7.12）定积分应用（二）+ 微分方程启动

- [ ] 做：定积分应用收尾 + 可分离变量基础题

#### 第11周（7.13-7.19）一阶线性微分方程
- [ ] 汤家凤基础课：一阶线性微分方程
- [ ] **通解公式必须刻进脑子**：`y=e^{-∫Pdx}[∫Q·e^{∫Pdx}dx+C]`

- [ ] 做：一阶线性基础题，练10道

#### 第12周（7.20-7.26）二阶常系数微分方程
- [ ] 汤家凤基础课：二阶常系数齐次 + 非齐次
- [ ] **齐次**：特征方程 r²+pr+q=0 → 三种情况（两不等实根/重根/共轭复根）
- [ ] **非齐次**（f(x)=Pₙ(x)e^{λx}型）：齐次通解 + 特解（待定系数法）
- [ ] k值判断（λ是不是特征根）是设特解的关键
- [ ] 做：二阶常系数基础题
- [ ] **月末检验**：二阶常系数齐次三种情况对答如流，非齐次特解能正确设出
