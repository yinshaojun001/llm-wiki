## 高等数学

### 一、函数、极限、连续（约10分，选择+填空）

#### 直观理解

极限回答一个问题："x无限靠近某个点时，f(x)无限靠近谁？"
连续就是"函数图像一笔画下来，没有洞、没有跳"。

作为工程师，你可以这样想：极限就是你不断 zoom in 函数图像，看到它趋于什么值。连续就是程序没有 throw exception —— 输入微变，输出也微变。

#### 核心概念

**1. 两个重要极限**（必须刻在脑子里）
- `lim(x→0) sinx/x = 1` —— 所有三角极限的祖宗
- `lim(x→∞) (1 + 1/x)^x = e` —— 所有 1^∞ 型的归宿

**2. 无穷小比较**（给两个趋于0的函数比谁更小）
- 高阶：A/B → 0，A比B小得快
- 同阶：A/B → 非零常数
- 等价：A/B → 1，可以互相替换 ← 这是解题利器
- 牢记等价无穷小：`x ~ sinx ~ tanx ~ arcsinx ~ arctanx ~ ln(1+x) ~ eˣ-1`，`1-cosx ~ x²/2`

**3. 间断点分类**
- 第一类：左右极限都存在（可去：左右相等≠函数值；跳跃：左右不等）
- 第二类：左右至少一个不存在（无穷、振荡）

#### 常见题型与解法模板

| 题型 | 解法 |
|---|---|
| 求极限 `0/0` 型 | 洛必达 / 等价无穷小替换 / 因式分解约分 |
| 求极限 `∞/∞` 型 | 洛必达 / 分子分母同除以最高次 |
| 求极限 `1^∞` 型 | 凑第二个重要极限 → `e^{lim指数×(底-1)}` |
| 求极限 `∞-∞` 型 | 通分 / 有理化 |
| 判断间断点类型 | 分别求左右极限 → 对照分类 |
| 无穷小比较 | 求商的极限，看结果是0/非零常数/1 |

#### 学到什么程度算过关
- 给一个极限题，30秒内判断类型，2分钟内做出来
- 等价无穷小替换不用查表
- 两个重要极限的变形（`sin3x/2x`、`(1+2/x)^x`）秒出

### 二、导数与微分（约15分，填空+计算）

#### 直观理解

导数是"瞬时变化率"——y随x变化的快慢。工程上就是：CPU利用率对时间的导数 = 负载变化速度。

微分是"微小变化量"——dy = f'(x)·dx。工程上就是：x变一点点，y大约变多少。

#### 核心公式

**基本求导公式**（必须条件反射，看到就能写）：
```
(C)'=0    (xⁿ)'=nxⁿ⁻¹    (eˣ)'=eˣ    (aˣ)'=aˣ·lna
(lnx)'=1/x    (log_a x)'=1/(x·lna)
(sinx)'=cosx    (cosx)'=-sinx    (tanx)'=sec²x    (cotx)'=-csc²x
(secx)'=secx·tanx    (cscx)'=-cscx·cotx
(arcsinx)'=1/√(1-x²)    (arccosx)'=-1/√(1-x²)
(arctanx)'=1/(1+x²)    (arccotx)'=-1/(1+x²)
```

**求导法则**：
- 四则运算：`(u±v)'=u'±v'`，`(uv)'=u'v+uv'`，`(u/v)'=(u'v-uv')/v²`
- 链式法则：`(f(g(x)))'=f'(g(x))·g'(x)` ← 复合函数求导的灵魂

**特殊求导类型**（数学二必考）：
- **隐函数求导**：方程两边同时对x求导，y是x的函数，y的导数是y'
- **参数方程求导**：`dy/dx = (dy/dt)/(dx/dt)`，二阶：`d²y/dx² = d(dy/dx)/dt ÷ dx/dt`

#### 常见题型与解法模板

| 题型 | 解法 |
|---|---|
| 复合函数求导 | 从外到内一层层剥，链式法则 |
| 隐函数求导 | 两边对x求导，y'项移到一边 |
| 参数方程一阶导 | dy/dx = y'(t)/x'(t) |
| 参数方程二阶导 | 先求一阶导，再对t求导，除以x'(t) |
| 高阶导数 | 逐阶求 / 找规律（eˣ的n阶=eˣ，sinx的n阶=sin(x+nπ/2)） |
| 求切线/法线方程 | 先求斜率(导数)，再点斜式 |

#### 工程直觉
- 导数 = 变化率，等价于你监控面板上的 QPS 变化曲线斜率
- 链式法则 = pipeline：CPU时间 = 指令数×CPI×时钟周期，每个因子变化都影响最终结果
- 极值点(导数=0) = 系统的最优配置点

#### 学到什么程度算过关
- 基本求导公式不用想，秒写
- 复合函数求导链式法则不出错
- 隐函数和参数方程求导各练20道，形成肌肉记忆

### 三、微分中值定理（约10分，证明题为主）

#### 直观理解

中值定理说：一条连续光滑的曲线，中间至少有一个点的切线斜率 = 两端连线的斜率。开车从北京到上海，平均速度100km/h，中间总有某一瞬间你的瞬时速度正好是100km/h。

#### 核心定理

| 定理 | 内容 | 用途 |
|---|---|---|
| **罗尔定理** | f(a)=f(b)，则存在ξ使f'(ξ)=0 | 证明存在导数为零的点 |
| **拉格朗日** | 存在ξ使f'(ξ)=(f(b)-f(a))/(b-a) | 证明不等式、存在性 |
| **柯西** | 存在ξ使[f(b)-f(a)]/[g(b)-g(a)]=f'(ξ)/g'(ξ) | 两个函数同时用 |
| **泰勒** | f(x)=f(x₀)+f'(x₀)(x-x₀)+f''(x₀)/2!(x-x₀)²+... | 函数近似、求极限 |

#### 常见题型与解法模板

**证明存在ξ使f'(ξ)=0**：找两点函数值相等 → 罗尔定理。关键是**构造原函数**，常见构造：
- 要证f'(ξ)+g'(ξ)=0 → 令F(x)=f(x)+g(x)
- 要证f'(ξ)+λf(ξ)=0 → 令F(x)=e^(λx)f(x)
- 要证ξf'(ξ)-nf(ξ)=0 → 令F(x)=f(x)/xⁿ

**证明不等式**：拉格朗日中值定理 → |f(b)-f(a)| = |f'(ξ)|·|b-a| ≤ M·|b-a|

**泰勒展开求极限**：展开到分子分母同阶，抵消后得结果。

#### 学中值定理的心态
证明题是数学二区分度的来源。**第一遍不求全会**，先把罗尔和拉格朗日的标准套路练熟（各练10道），柯西和泰勒二轮再补。考试时证明题做不出来不慌，10分丢了还有140。

#### 学到什么程度算过关
- 罗尔定理：能自己构造原函数
- 拉格朗日：能用来证明简单不等式
- 不要求所有证明题都会做

### 四、不定积分（约10分，计算题基础）

#### 直观理解

不定积分是求导的逆运算。知道变化率，反推原函数。工程上：知道速度变化曲线，反推行驶距离。

不定积分本身不直接考大题，但它是定积分、微分方程的基础。**积不出来，后面全卡死。**

#### 核心方法

**第一类换元法（凑微分）** ← 最重要的基本功
```
核心思想：∫f(g(x))g'(x)dx = ∫f(u)du，其中u=g(x)
常见凑微分模式：
∫f(ax+b)dx → u=ax+b, du=a·dx
∫f(eˣ)eˣdx → u=eˣ
∫f(lnx)/x dx → u=lnx
∫f(sinx)cosx dx → u=sinx
∫f(cosx)sinx dx → u=cosx（注意负号）
∫f(tanx)sec²x dx → u=tanx
∫f(arctanx)/(1+x²)dx → u=arctanx
```

**第二类换元法**（根号下的套路）：
- √(a²-x²) → 令x=a·sint
- √(a²+x²) → 令x=a·tant
- √(x²-a²) → 令x=a·sect

**分部积分法**：
```
∫u dv = uv - ∫v du
口诀："反对幂指三"——左边优先做u（反三角>对数>幂函数>指数>三角）
典型：
∫x·eˣdx → u=x, dv=eˣdx
∫x·sinx dx → u=x, dv=sinx dx（要分部两次）
∫lnx dx → u=lnx, dv=dx
∫eˣsinx dx → 分部两次后会回到原积分，移项解方程
```

**有理函数积分**：
- 假分式→多项式+真分式（长除法）
- 真分式分母因式分解后，拆成部分分式之和

#### 常见题型与解法模板

| 题型 | 先试什么 |
|---|---|
| 被积函数有复合结构 | 第一类换元（凑微分） |
| 被积函数有√(a²±x²) | 第二类换元（三角代换） |
| 两种不同类型函数相乘 | 分部积分 |
| 有理分式 | 部分分式拆开 |
| 三角函数有理式 | 万能代换(t=tan(x/2))或凑sin/cos |

#### 学到什么程度算过关
- 凑微分法：10道基本题不错
- 分部积分：∫x·eˣdx、∫x·sinx dx、∫lnx dx 三种经典模式闭卷写出
- 第二类换元：三种根号下知道对应代换

### 五、定积分（约15分，填空+计算大题）

#### 直观理解

定积分 = 曲线下的面积。不定积分给你一个函数族(F(x)+C)，定积分给你一个数。

工程上：QPS对时间积分 = 总请求量。CPU利用率对时间积分 = 总CPU时间。

#### 核心公式：牛顿-莱布尼茨公式
```
∫[a,b] f(x)dx = F(b) - F(a)，其中F'(x)=f(x)
```
这是连接不定积分和定积分的桥梁。不定积分是工具，定积分是应用。

#### 核心技巧

**变上限函数**（数学二必考）：
```
Φ(x) = ∫[a,x] f(t)dt，则 Φ'(x) = f(x)
变上限是上限的函数，求导就把上限代入
如果上限是g(x)：Φ'(x) = f(g(x))·g'(x) ← 链式法则
```

**定积分换元**：**换元必须换限！**x换成t，积分上下限也要从x的范围换成t的范围。

**定积分分部积分**：`∫[a,b] u dv = uv|[a,b] - ∫[a,b] v du`

**对称区间奇偶性**（省时间利器）：
- f(x)是奇函数，积分区间关于原点对称 → 积分为0
- f(x)是偶函数，积分区间关于原点对称 → 积分=2×∫[0,a]

**广义积分**（无穷区间/无界函数）：正常积分算出来，然后取极限。

#### 常见题型与解法模板

| 题型 | 解法 |
|---|---|
| 普通定积分 | 牛顿-莱布尼茨，先积再代 |
| 变上限函数求导 | 上限代入×上限导数 |
| 对称区间积分 | 先看奇偶，奇函数直接0 |
| 含绝对值的定积分 | 分区间去绝对值 |
| 广义积分 | 先算正常积分，再取极限 |

#### 学到什么程度算过关
- 变上限求导秒出
- 定积分换元不忘记换限
- 对称区间第一时间反应奇偶

### 六、定积分应用（约10分，计算大题）

#### 直观理解

积分就是把无穷多小份加在一起。面积 = 把无穷多小长方形面积加起来。体积 = 把无穷多小薄片体积加起来。

#### 必考三类

**1. 平面图形面积**
```
直角坐标：
  X型区域：A = ∫[a,b] [上函数-下函数]dx
  Y型区域：A = ∫[c,d] [右函数-左函数]dy

极坐标：A = 1/2 ∫[α,β] r²(θ)dθ

选择X还是Y型：看区域是被竖条还是横条完整覆盖
```

**2. 旋转体体积**
```
绕x轴旋转：V = π∫[a,b] y²dx（圆盘法）
绕y轴旋转：V = π∫[c,d] x²dy

柱壳法（绕y轴，用x积分）：V = 2π∫[a,b] x·y·dx
```

**3. 弧长**
```
直角坐标：s = ∫[a,b] √(1+y'²)dx
参数方程：s = ∫[α,β] √(x'²+y'²)dt
```

#### 常见题型与解法模板

| 题型 | 步骤 |
|---|---|
| 求两曲线围成面积 | ①画草图 ②求交点 ③判断X/Y型 ④∫(上-下)dx |
| 绕坐标轴旋转体积 | ①确定旋转轴 ②选圆盘法/柱壳法 ③∫ |
| 绕平行于坐标轴的直线旋转 | 平移坐标系 → 圆盘法 |

#### 学到什么程度算过关
- 给两条曲线，能画出草图，写出面积公式
- 绕x轴和绕y轴的体积公式不混
- 弧长公式记住（考得少但会考）

### 七、微分方程（约12分，大题）

#### 直观理解

微分方程 = 含有未知函数及其导数的方程。解微分方程 = 从"变化规律"反推出"函数本身"。

工程上：冷却定律（温度变化率正比温差）→ 微分方程 → 解出温度随时间的变化函数。种群增长、RC电路充放电、放射性衰变，全是微分方程。

#### 一阶微分方程

**可分离变量**（最基础）：
```
dy/dx = f(x)·g(y)
→ dy/g(y) = f(x)dx
→ 两边积分
```

**一阶线性**（必考大题）：
```
标准形式：y' + P(x)y = Q(x)
通解公式：y = e^{-∫Pdx} [∫Q·e^{∫Pdx}dx + C]

步骤：
① 化成标准形式：y'系数必须为1
② 找出P(x)，计算e^{∫Pdx}
③ 套公式
④ 有初始条件就代入定C
```

**齐次方程**（比较少考）：`dy/dx = f(y/x)`，令u=y/x代换。

#### 二阶常系数线性微分方程（数学二必考大题）

**齐次**：y'' + py' + qy = 0
```
特征方程：r² + pr + q = 0

情况1：两不等实根 r₁≠r₂
  y = C₁e^{r₁x} + C₂e^{r₂x}

情况2：两相等实根 r₁=r₂=r
  y = (C₁ + C₂x)e^{rx}

情况3：共轭复根 r = α ± βi
  y = e^{αx}(C₁cosβx + C₂sinβx)
```

**非齐次**：y'' + py' + qy = f(x)
```
通解 = 齐次通解 + 非齐次特解

f(x) = Pₙ(x)e^{λx} 型：
  设特解 y* = xᵏ·Qₙ(x)e^{λx}
  k = 0（λ不是特征根）/ 1（λ是单根）/ 2（λ是重根）
  Qₙ(x)是待定的n次多项式

f(x) = e^{αx}(Pₗ(x)cosβx + Pₙ(x)sinβx) 型：
  设特解 y* = xᵏ·e^{αx}(Rₘcosβx + Sₘsinβx)
  其中m = max(l,n)
  k = 0（α+βi不是特征根）/ 1（α+βi是特征根）
```

#### 常见题型与解法模板

| 题型 | 解法 |
|---|---|
| 可分离变量 | 分离→积分→化简 |
| 一阶线性 | 化标准型(y'+Py=Q)→套公式 |
| 二阶常系数齐次 | 写特征方程→求根→按三种情况写通解 |
| 二阶常系数非齐次 | 齐次通解+根据f(x)类型设特解→待定系数 |
| 应用题（几何/物理） | 根据条件列微分方程→求解→初始条件定常数 |

#### 学到什么程度算过关
- 一阶线性通解公式刻在脑子里
- 二阶常系数齐次三种情况对答如流
- 非齐次特解设法的k值判断（λ是不是特征根）

### 八、多元函数微分学（约15分，大题）

#### 直观理解

一元函数y=f(x)的导数是dy/dx。多元函数z=f(x,y)有偏导数∂z/∂x（y固定，z对x的变化率）和∂z/∂y（x固定，z对y的变化率）。

工程上：服务延迟z是QPS(x)和线程池大小(y)的函数，∂z/∂x告诉我们QPS对延迟的影响（线程池固定），∂z/∂y告诉我们线程池对延迟的影响（QPS固定）。

#### 核心概念

**偏导数**：对x求偏导时，y视为常数（反之亦然）

**全微分**：dz = (∂z/∂x)dx + (∂z/∂y)dy —— 两个方向的变化叠加

**链式法则**（多元复合函数求导，必考）：
```
z=f(u,v), u=u(x,y), v=v(x,y)
∂z/∂x = (∂z/∂u)(∂u/∂x) + (∂z/∂v)(∂v/∂x)
∂z/∂y = (∂z/∂u)(∂u/∂y) + (∂z/∂v)(∂v/∂y)
画变量关系图，从z到目标变量每一条路径乘起来，所有路径加起来。
```

**隐函数求导**（一个方程F(x,y)=0确定y=f(x)）：
```
dy/dx = -F_x/F_y（公式法）
或两边对x求导，y是x的函数（直接法）
```

#### 多元函数的极值

**无条件极值**：
```
① 求驻点：∂z/∂x=0, ∂z/∂y=0
② 判别：A=∂²z/∂x², B=∂²z/∂x∂y, C=∂²z/∂y²
   AC-B²>0且A>0 → 极小值
   AC-B²>0且A<0 → 极大值
   AC-B²<0 → 不是极值
   AC-B²=0 → 不确定，需要其他方法
```

**条件极值（拉格朗日乘数法）** ← 数学二必考大题：
```
问题：在约束φ(x,y)=0下，求z=f(x,y)的极值

步骤：
① 构造拉格朗日函数：L(x,y,λ) = f(x,y) + λ·φ(x,y)
② 求偏导=0：
   ∂L/∂x = 0
   ∂L/∂y = 0
   ∂L/∂λ = 0（即φ(x,y)=0）
③ 解方程组得(x,y)
④ 代入f(x,y)得极值
```

#### 常见题型与解法模板

| 题型 | 解法 |
|---|---|
| 求偏导数 | 对谁求偏导，其他变量当常数 |
| 复合函数偏导 | 画路径图→链式法则 |
| 隐函数求导 | F(x,y)=0则dy/dx=-F_x/F_y |
| 无条件极值 | 驻点→AC-B²判别 |
| 条件极值 | 拉格朗日乘数法 |

#### 学到什么程度算过关
- 偏导计算不出错（最容易扣分的其实是把变量搞混）
- 复合函数链式法则画图后不错
- 拉格朗日乘数法标准化步骤，练10道形成肌肉记忆

### 九、二重积分（约15分，大题）

#### 直观理解

定积分是曲线下面积。二重积分是曲面下的体积。把平面区域切成无穷多小方块，每个小方块上的函数值×小方块面积，全加起来。

#### 核心技能

**在直角坐标下计算**（最重要）：
```
∬_D f(x,y)dσ = ∫dx∫f(x,y)dy 或 ∫dy∫f(x,y)dx

X型区域（先y后x）：D = {(x,y)|a≤x≤b, y₁(x)≤y≤y₂(x)}
∬_D f(x,y)dσ = ∫[a,b]dx ∫[y₁(x),y₂(x)] f(x,y)dy

Y型区域（先x后y）：D = {(x,y)|c≤y≤d, x₁(y)≤x≤x₂(y)}
∬_D f(x,y)dσ = ∫[c,d]dy ∫[x₁(y),x₂(y)] f(x,y)dx
```

**交换积分次序**（必考）：
```
有时先y后x积不出来，需要换成先x后y。步骤：
① 从原积分画出积分区域D
② 用另一种方式描述D的边界
③ 写出新次序的积分
```

**在极坐标下计算**：
```
当积分区域是圆/扇形/圆环时，用极坐标：
x = r·cosθ, y = r·sinθ, dσ = r·dr·dθ
∬_D f(x,y)dσ = ∫[α,β]dθ ∫[r₁(θ),r₂(θ)] f(rcosθ,rsinθ)·r·dr
**不要忘记那个 r！**这是极坐标变换最容易被扣分的地方。
```

#### 常见题型与解法模板

| 题型 | 解法 |
|---|---|
| 直角坐标二重积分 | 画区域→定积次序→逐层积分 |
| 交换积分次序 | 从累次积分还原区域→重新描述→换次序 |
| 极坐标二重积分 | 判断区域是否圆形→换元→别忘了r |
| 含绝对值的二重积分 | 分区域去绝对值 |
| 利用对称性 | 奇函数在对称区域积分为0 |

#### 工程直觉
- 二重积分 = MapReduce：把大区域切分成小块，每块独立算，最后汇总
- 交换积分次序 = 改变循环嵌套顺序（先按行遍历还是先按列遍历）

#### 学到什么程度算过关
- 看到积分区域能画出草图
- 直角坐标和极坐标的选择有判断力
- 交换积分次序练10道，形成画图习惯

## 线性代数

> 线代约30分，但性价比极高——题型固定，练熟就能拿满分。

### 一、行列式（约3分，填空）

#### 直观理解

行列式是一个数，用来判断方阵是否"可逆"。|A|≠0 → 可逆；|A|=0 → 不可逆（奇异）。

#### 核心性质（算行列式的捷径）

```
① |A^T| = |A|
② 交换两行→变号
③ 某行乘k→行列式乘k
④ 某行加到另一行→值不变 ← 消元法的基础
⑤ |AB| = |A|·|B|
⑥ 上三角/下三角→主对角元乘积
```

#### 计算方法

- **2阶/3阶**：直接算。3阶用沙路法或按行展开
- **4阶及以上**：用性质消成上三角，再乘对角线
- 很多元素相同/行和相等→先化简再算

#### 特殊行列式（数学二爱考）
- 范德蒙行列式：每行是等比数列
- 爪型行列式：只有主对角+第一行/列有元素

---

### 二、矩阵（约5分，填空+计算）

#### 直观理解

矩阵就是一张数表。工程上每个表格、每个线性变换都是矩阵。矩阵乘法 = 组合变换。

#### 核心运算

**矩阵乘法**：A(m×n)·B(n×p)=C(m×p)，Cij = A第i行·B第j列（点乘）。**乘法不满足交换律AB≠BA**。

**转置**：(A^T)ij = Aji。性质：(AB)^T = B^T·A^T（顺序反转）

**逆矩阵**：A·A^(-1) = I。口诀：`A^(-1) = A*/|A|`（伴随矩阵法，考试算低阶用）
- 2阶求逆：`[[a,b],[c,d]]^(-1) = 1/(ad-bc)·[[d,-b],[-c,a]]`

**矩阵的秩** r(A)：非零子式的最高阶数 = 行阶梯形中非零行数。秩是矩阵的"信息量"。

**矩阵方程**：
- AX=B → X=A^(-1)B
- XA=B → X=BA^(-1)
- AXC=B → 左右同时处理

#### 分块矩阵
把大矩阵切成小块，对块做运算和普通矩阵一样，但块乘法不满足交换律。分块对角阵的逆=各块分别求逆。

---

### 三、向量组（约5分，计算+证明）

#### 直观理解

向量组：一堆向量放在一起。核心问题是：有没有冗余（线性相关）？能张成多大空间（秩）？

工程上：特征向量组就是降维变换的方向集合。

#### 核心概念

**线性相关 vs 线性无关**：
- 线性相关：存在不全为零的系数k₁,k₂,...，使得k₁α₁+k₂α₂+...=0（有冗余）
- 线性无关：只有全为零的系数才满足，即所有向量都"独立"
- 判别：向量个数 > 维数 → 必相关；向量排成矩阵，秩 r < 向量个数 → 相关

**极大无关组**：从向量组中挑出几个线性无关的向量，且任意向量都可由它们表出。极大无关组 = 向量组的"骨架"。

**向量组的秩** = 极大无关组中向量的个数 = 矩阵的秩（排成矩阵后做行变换）

**线性表出**：β能否由α₁,α₂,...表出 → 解方程组 → 有解就是能。**唯一表出**要求α们线性无关。

---

### 四、线性方程组（约6分，大题）

#### 直观理解

Ax=b。A是系数矩阵，x是要求的未知数，b是结果。有解吗？解唯一吗？这就是线性方程组的全部。

工程上：每处都在解线性方程组——最小二乘拟合、PageRank、电路分析。

#### 解的判定（最重要的基础）

```
Ax=b 有解 ⇔ r([A|b]) = r(A)（增广矩阵的秩 = 系数矩阵的秩）
  有解且 r(A)=n（未知数个数）→ 唯一解
  有解且 r(A)<n → 无穷多解（含 n-r(A) 个自由变量）

Ax=0（齐次）总是有解（x=0是解，称零解/平凡解）
  有非零解 ⇔ r(A)<n
```

#### 解的求法

**齐次Ax=0**：
① 系数矩阵→行阶梯形
② 确定自由变量（不在主元列的变量）
③ 令自由变量依次取(1,0,0...), (0,1,0...)...，回代求出主元变量的值
④ 通解 = 自由变量的任意线性组合

**非齐次Ax=b**：
① 增广矩阵[A|b]→行阶梯形
② 先判断有无解（r(A) 是否等于 r([A|b])）
③ 求一个特解（令自由变量=0，回代求出其他变量）
④ 求对应齐次的通解
⑤ 通解 = 特解 + 齐次通解

#### 常见题型与解法模板

| 题型 | 关键 |
|---|---|
| 判断解的存在唯一性 | 算r(A)和r([A|b]) |
| 求齐次通解 | 行阶梯→自由变量→基础解系 |
| 求非齐次通解 | 特解+齐次通解 |
| 含参数的方程组讨论 | 参数不同值时，r(A)可能不同→分类讨论 |

---

### 五、特征值与特征向量（约6分，大题）

#### 直观理解

矩阵A作用在向量x上得到Ax。大多数向量被A改变方向和长度。但有些特殊向量，A作用后方向不变，只被伸缩：**Ax = λx**。这就是特征向量x和特征值λ。

工程上：特征向量 = 系统的主模式（振动的主振型、数据的主成分方向）。PageRank就是求Google矩阵最大特征值对应的特征向量。

#### 计算步骤

**求特征值**：
```
|A - λI| = 0 → 特征方程 → 解出λ₁, λ₂, ...
```
3阶矩阵的特征方程展开：`λ³ - tr(A)·λ² + (三个二阶主子式之和)·λ - |A| = 0`

**求特征向量**：
对每个λᵢ，解(A-λᵢI)x=0，求出的非零解就是λᵢ的特征向量。

**性质**：
- 不同特征值的特征向量线性无关
- tr(A) = Σλᵢ（迹=特征值之和）
- |A| = Πλᵢ（行列式=特征值之积）

#### 相似对角化

如果A有n个线性无关的特征向量（等价于每个特征值的几何重数=代数重数），则A可以相似对角化：
```
P⁻¹AP = Λ = diag(λ₁,λ₂,...,λₙ)
其中P是由特征向量按列排成的矩阵
```

**判定**：实对称矩阵一定可以正交相似对角化——这是二次型的基础。

#### 常见题型与解法模板

| 题型 | 步骤 |
|---|---|
| 求特征值特征向量 | |A-λE|=0→λ→解(A-λE)x=0 |
| 判断能否对角化 | 看每个λ的几何重数是否等于代数重数 |
| 相似对角化 | P⁻¹AP=Λ，写P和Λ |
| 由特征值反推矩阵 | 用tr(A)=Σλ和|A|=Πλ |

### 六、二次型（约5分，大题）

#### 直观理解

二次型 = 只有二次项（x₁², x₁x₂, x₂², ...）的多元多项式。写成矩阵形式：f = x^T A x，其中A是实对称矩阵。

工程上：二次型用来判定多元函数的极值（Hessian矩阵的正定性）。协方差矩阵就是二次型。

#### 核心内容

**化二次型为标准型**（三种方法，推荐正交变换法）：
```
目标：把 f = x^T A x 变成 f = λ₁y₁² + λ₂y₂² + ...（只有平方项，没有交叉项）

正交变换法（最常用）：
① 写出二次型矩阵A（对称矩阵）
② 求A的特征值λ₁,λ₂,...,λₙ
③ 求对应的正交单位特征向量（施密特正交化+单位化）
④ 正交变换 x=Qy（Q是正交矩阵，即Q^T=Q^(-1)）
⑤ 得标准型：f = λ₁y₁² + λ₂y₂² + ... + λₙyₙ²
```

**正定二次型**：
```
f(x₁,x₂,...,xₙ)>0 对所有非零x成立
判定（等价条件，全都要记住）：
① 所有特征值>0
② 所有顺序主子式>0
③ 正惯性指数=n
```

**合同**：如果存在可逆矩阵C使得B=C^T AC，则A与B合同。合同变换改变矩阵但不改变正负特征值的个数（惯性定理）。

#### 常见题型与解法模板

| 题型 | 步骤 |
|---|---|
| 化标准型（正交变换） | 求特征值→特征向量→正交化→写标准型 |
| 化标准型（配方法） | 逐项配方，消交叉项 |
| 判定正定性 | 特征值法或顺序主子式法 |
| 已知标准型反推参数 | 用惯性定理（正负特征值个数不变） |

## 必背公式卡片

### 导数表
```
(C)'=0    (xⁿ)'=nxⁿ⁻¹    (eˣ)'=eˣ    (aˣ)'=aˣlna
(lnx)'=1/x    (log_a x)'=1/(x·lna)
(sinx)'=cosx    (cosx)'=-sinx    (tanx)'=sec²x    (cotx)'=-csc²x
(secx)'=secx·tanx    (cscx)'=-cscx·cotx
(arcsinx)'=1/√(1-x²)    (arccosx)'=-1/√(1-x²)
(arctanx)'=1/(1+x²)    (arccotx)'=-1/(1+x²)
```

### 积分表
```
∫xⁿdx = xⁿ⁺¹/(n+1)+C (n≠-1)    ∫1/x dx = ln|x|+C
∫eˣdx = eˣ+C    ∫aˣdx = aˣ/lna+C
∫sinx dx = -cosx+C    ∫cosx dx = sinx+C
∫tanx dx = -ln|cosx|+C    ∫cotx dx = ln|sinx|+C
∫secx dx = ln|secx+tanx|+C    ∫cscx dx = ln|cscx-cotx|+C
∫1/(1+x²)dx = arctanx+C    ∫1/√(1-x²)dx = arcsinx+C
∫1/(a²+x²)dx = (1/a)arctan(x/a)+C
∫1/√(a²-x²)dx = arcsin(x/a)+C
∫1/(x²-a²)dx = (1/2a)ln|(x-a)/(x+a)|+C
```

### 微分方程通解
```
一阶线性: y'+P(x)y=Q(x)
  → y = e^{-∫Pdx}[∫Q·e^{∫Pdx}dx+C]

二阶常系数齐次: y''+py'+qy=0, 特征方程 r²+pr+q=0
  两不等实根: y = C₁e^{r₁x}+C₂e^{r₂x}
  两相等实根: y = (C₁+C₂x)e^{rx}
  共轭复根(r=α±βi): y = e^{αx}(C₁cosβx+C₂sinβx)

二阶常系数非齐次: y''+py'+qy=Pₙ(x)e^{λx}
  设特解 y* = xᵏ·Qₙ(x)e^{λx}
  k=0(λ不是特征根)/k=1(λ是单根)/k=2(λ是重根)
```

## 避坑指南

1. **计算错误是数学二扣分第一原因**。草稿纸规范：每道题一行，步骤写完整，不跳步
2. **不要贪多**：1000题做透800题 > 1800题做一半。做完的每道题都要搞懂
3. **真题是核心**：近15年真题至少刷两遍。模拟题可以不写，真题必须写
4. **线代不难但爱丢分**：特征值、二次型大题步骤固定，每种练10道就稳
5. **定积分/二重积分计算是分数大头**：每天做2道纯计算保持手感
6. **第一遍不强求全会**：中值定理证明、数列极限证明可以先放，二轮再补
7. **公式不要死记**：导数公式靠做题自然记住，积分公式需要刻意背
8. **真题第一遍会很惨**：第一套可能只有60分，这是正常的。学到第四阶段再碰真题
