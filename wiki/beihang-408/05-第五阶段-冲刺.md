# 第五阶段：全科冲刺

> 2027年9月 - 12月20日左右（约15周）| 408真题二刷 + 数学模考 + 政治英语

---

## 九月（4周）

### 408
- [ ] 真题二刷（2016-2024），每周2套，限时3小时
- [ ] 这次重点：不是会不会做，是**能不能在规定时间做完**
- [ ] 选择题控制在80分钟内，大题每道15-20分钟

### 数学二
- [ ] **真题错题二刷**：只做一刷错题，做完立刻对照答案
- [ ] **模拟题**：张宇8套卷 或 李林6套卷，每周1-2套，限时3小时
- [ ] 每次模考后统计三类丢分：
  - 计算错误（草稿纸跳步→罚每天10道纯计算）
  - 知识点不会（回归[[14-科目-数学二]]该章节重学）
  - 时间不够（提高计算速度，选填每道<3分钟）
- [ ] 目标：模拟题均分>95

### 政治 — 启动
- [ ] 徐涛强化课（通勤+午休听，每天45分钟）
- [ ] 九月听完马原+史纲
- [ ] 马原要理解（唯物辩证法+政治经济学），史纲记时间线

### 英语二
- [ ] 真题阅读（每天1篇，15分钟/篇）
- [ ] 午休背单词（墨墨/不背单词App，15min）
