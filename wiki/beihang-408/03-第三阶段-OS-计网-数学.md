# 第三阶段：OS + 计网 + 数学收尾

> 2027年1月-4月（17周）| 你有体感的科目，快速收割

---

## 一月（4周）：OS进程管理

### 第1周（1.4-1.10）
- [ ] 王道OS 第一章 概述：定义/特征(并发共享虚拟异步)、发展历程
- [ ] 内核态/用户态、中断/异常、系统调用（与库函数区别）
- [ ] 做：第一章全选

### 第2周（1.11-1.17）
- [ ] 第二章 2.1 进程与线程
  - 进程状态转换(就绪/运行/阻塞)、PCB
  - 线程 vs 进程 ← 你天天写 `new Thread()`
  - 进程通信：共享内存/消息传递/管道/信号
- [ ] 做：2.1选

### 第3周（1.18-1.24）
- [ ] 2.2 CPU调度
  - 七种算法：FCFS / SJF / 优先级 / 高响应比 / RR / 多级队列 / 多级反馈队列
  - 周转时间 = 完成时间 - 到达时间；带权周转时间 = 周转时间/运行时间
- [ ] 做：2.2选择 + 大1、2

### 第4周（1.25-1.31）
- [ ] 2.3 **同步与互斥** ← OS大题最高频
  - 信号量PV操作
  - **三个经典问题必须手写**：
    1. 生产者-消费者（单/多生产者，单/多缓冲区）
    2. 读者-写者（读者优先/写者优先）
    3. 哲学家进餐（防死锁：限制同时进餐人数/奇数先左偶数先右）
- [ ] 2.4 死锁：4条件 + 银行家算法
- [ ] 做：2.3-2.4全选 + PV大题

**从Java看**：`synchronized` = 互斥锁，`wait()/notify()` = PV操作，`ThreadPoolExecutor` = 线程池

## 二月（4周）：OS内存 + 文件 + IO

### 第5-6周（2.1-2.14）
- [ ] 第三章 3.1 内存管理基础
  - 连续分配：首次/最佳/最坏/邻近适应
  - 分页：页表、地址变换（逻辑地址→物理地址）、TLB
  - 分段 + 段页式
- [ ] 3.2 虚拟内存
  - 请求分页、页框分配与回收（2025新增考点）
  - 页面置换算法：**OPT/LRU/FIFO/Clock/改进Clock** ← 手写模拟
  - 抖动、工作集
- [ ] 做：第三章大题（页表地址转换 + 页面置换）

**从Java看**：OOM、`-Xmx` = 虚拟内存。`LinkedHashMap` accessOrder=true = LRU

### 第7周（2.15-2.21）
- [ ] 第四章 文件管理
  - inode、目录结构、硬链接/软链接
  - 文件物理结构：连续/链接/索引/混合索引分配 ← 大题计算
  - 空闲空间管理：位示图法（考最多）
- [ ] 做：第四章选择+大题（混合索引 + 位示图）

**从Linux看**：`ls -i` = inode号，`ln` vs `ln -s` = 硬链接/软链接

### 第8周（2.22-2.28）
- [ ] 第五章 I/O管理
  - 磁盘调度：FCFS/SSTF/SCAN/C-SCAN/LOOK/C-LOOK
  - SPOOLing
  - 单缓冲/双缓冲时间计算
- [ ] 做：第五章大题（磁盘调度）

## 三月（5周）：计网

### 第9周（3.1-3.7）
- [ ] 王道计网 第一章 概述：OSI七层/TCPIP四层、性能指标
- [ ] 第二章 物理层：奈奎斯特/香农公式、编码（曼彻斯特/差分曼彻斯特）
- [ ] 做：第一、二章选择

### 第10周（3.8-3.14）
- [ ] 第三章 数据链路层
  - 组帧、CRC计算、海明码
  - CSMA/CD（以太网）、最小帧长计算
  - 交换机自学习、STP、VLAN
- [ ] 做：第三章选择+大题（CSMA/CD + 交换机转发）

### 第11周（3.15-3.21）
- [ ] 第四章 网络层
  - **IP子网划分** ← 每年必考：给定IP/CIDR，划分子网，计算子网地址/广播地址/可用IP
  - **路由聚合**
  - ARP、DHCP、ICMP、NAT
- [ ] 做：第四章选择（子网划分专项练10道）

**从日常看**：`ip addr` = 你机器的IP和子网掩码，`ping` = ICMP，`traceroute` = TTL

### 第12-13周（3.22-4.4）
- [ ] 第四章续 路由协议：RIP(距离向量/UDP/520)、OSPF(链路状态/IP/89)、BGP(路径向量/TCP/179)
- [ ] IPv6、IP组播、移动IP
- [ ] 第五章 传输层：
  - **TCP** ← 计网大题最高频
  - 三次握手(SYN→SYN-ACK→ACK)、四次挥手(FIN→ACK→FIN→ACK)
  - TCP状态转换图（TIME_WAIT为什么2MSL？）
  - 拥塞控制：慢开始/拥塞避免/快重传/快恢复，ssthresh/cwnd/rwnd演变图
  - UDP：首部8字节，无连接
- [ ] 做：第四、五章大题

**从日常看**：`curl -v` 看到三次握手，`ss -tan` 看TCP状态

## 数学二（详细安排）

> 目标：高数+线代强化刷题 → 真题一刷启动
> 第三阶段数学从"学知识"转为"练考试"
> 每章对照 [[14-科目-数学二]] 看详细讲解

### 一月（4周）：高数强化 + 线代强化

#### 第1周（1.4-1.10）极限+导数 强化
- [ ] 汤家凤强化课：极限+连续+导数
- [ ] 1800题强化篇：极限章节（每天10道）
- [ ] **重点**：数列极限证明（单调有界、夹逼），之前一轮可能没弄懂，强化阶段补
- [ ] 导数定义题（给极限式子，判断是什么的导数）
- [ ] 错题本：每种极限类型的错题各收录3道

#### 第2周（1.11-1.17）中值定理 强化
- [ ] 汤家凤强化课：中值定理
- [ ] 罗尔定理构造原函数练习（每天2道）
- [ ] 拉格朗日证明不等式（每天2道）
- [ ] 泰勒展开求极限（展开到同阶）
- [ ] 做：1800题中值定理强化篇
- [ ] **不强求全会**：中值定理证明题是区分度的来源，能做对60%就很好

#### 第3周（1.18-1.24）积分 强化
- [ ] 汤家凤强化课：不定积分+定积分
- [ ] 1800题积分强化篇（每天15道，重点练计算速度和准确率）
- [ ] 定积分综合题（变上限+换元+分部同时用）
- [ ] 广义积分（无穷区间+无界函数）
- [ ] **检验**：不定积分随机10道，8道以上正确且每道<3分钟

#### 第4周（1.25-1.31）定积分应用+微分方程 强化
- [ ] 汤家凤强化课：定积分应用+微分方程
- [ ] 旋转体体积综合（圆盘法vs柱壳法的选择）
- [ ] 微分方程应用题（几何+物理背景）
- [ ] 二阶非齐次特解法，两种f(x)类型都要练
- [ ] 做：1800题定积分应用+微分方程强化篇
- [ ] **月末检验**：定积分应用大题+微分方程大题各3道，步骤完整不出错

### 二月（4周）：多元微积分强化 + 线代强化

#### 第5-6周（2.1-2.14）多元微分+二重积分 强化

- [ ] 拉格朗日乘数法综合题（练10道，各种约束条件）

- [ ] 做：1800题多元微分+二重积分强化篇

#### 第7周（2.15-2.21）线代强化（一）

- [ ] 行列式计算提速（消元法+特殊行列式识别）

- [ ] **检验**：含参数方程组讨论题，能完整分类（有解/无解/唯一解/无穷解）

#### 第8周（2.22-2.28）线代强化（二）
- [ ] 汤家凤线代强化课：特征值+二次型

### 三月（5周）：数学二真题一刷

#### 第9-13周（3.1-4.4）真题一刷
- [ ] **2013-2024年数学二真题，共12套，每周2-3套**
- [ ] 每套严格限时3小时（和真实考试一样）
- [ ] 做完后精析：

### 四月（3周）：真题一刷收尾 + 错题整理
