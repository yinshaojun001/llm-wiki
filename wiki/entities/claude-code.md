# Claude Code

**类型**：AI 编程 CLI 工具
**开发商**：Anthropic
**定位**：当下 Agent 能力上限最高、生态最成熟的 CLI 工具之一

## 核心能力

- **Auto Mode**（2026-03-27）：基于分类器的权限自动化，自主执行具备产品级入口
- **Plan Mode**：执行前先规划，可审查再行动
- **AGENTS.md**：项目级 AI 配置标准，正推动跨工具迁移性
- **MCP 支持**：Model Context Protocol 工具调用

## 2026-03 重要事件

| 事件 | 影响 |
|---|---|
| Auto Mode 发布 | 自主执行进入产品 |
| `git reset --hard` bug | 安全边界问题暴露 |
| Max 幽灵限流危机 | 计费透明度变行业议题 |

## 竞争优势与风险

- **优势**：能力上限最高，[[concepts/agents-md]] 标准推动者
- **风险**：能力越强，用户越在意可控性；信任成本随自动化程度上升
