## 第一阶段：数据结构 + 数学二启动（2026.05 - 2026.08）

**目标**：数据结构从会用→会考，数学二进入学习状态

### 数据结构（主攻）

```
5月：线性表 + 栈队列 + 串
     ├─ 从ArrayList/LinkedList源码反推顺序表/链表
     ├─ 手写基本操作（插入、删除、反转）
     └─ 王道课后选择题

6月：树与二叉树
     ├─ TreeMap源码反推二叉搜索树
     ├─ 四种遍历（前中后层）手写递归+非递归
     ├─ 哈夫曼树、并查集
     └─ 重点：树是408数据结构大题高频考点

7月：图 + 查找
     ├─ BFS/DFS手写，最小生成树，最短路径
     ├─ 哈希表（从HashMap源码反推，你天天用）
     ├─ 二分查找、B树/B+树（MySQL索引天天碰）
     └─ 哈希表冲突解决是高频考点

8月：排序 + 数据结构收尾
     ├─ 八大排序手写（快排/归并/堆排序是重点）
     ├─ Arrays.sort()源码反推TimSort
     ├─ 时间复杂度对比表烂熟于心
     └─ 王道数据结构第一遍完成
```

**检验标准**：随机一道王道数据结构大题，30分钟内写出正确代码。

### 数学二（启动）

> 详细教学内容和公式参考 [[14-科目-数学二]]，每周安排见 [[01-第一阶段-DS-数学]]

```
5月：函数极限连续 + 导数与微分
    ├─ 两个重要极限、等价无穷小替换、间断点分类
    ├─ 基本求导公式（每天默写）、复合函数求导（链式法则）
    ├─ 隐函数求导、参数方程求导
    └─ 汤家凤基础课 + 课后基础题

6月：微分中值定理 + 不定积分 + 定积分
    ├─ 罗尔定理（构造原函数）、拉格朗日（证明不等式）
    ├─ 不定积分三大法：凑微分、第二类换元、分部积分
    ├─ 定积分：牛顿-莱布尼茨 + 变上限求导 + 换元必换限
    └─ 汤家凤基础课 + 1800题基础篇

7月：定积分应用 + 微分方程
    ├─ 面积（X型/Y型）、旋转体体积（圆盘法/柱壳法）、弧长
    ├─ 可分离变量 + 一阶线性（通解公式刻脑子里）
    └─ 二阶常系数齐次+非齐次（特征方程→三种情况）

8月：多元微分 + 二重积分 + 高数收尾
    ├─ 偏导数、全微分、复合函数链式法则
    ├─ 拉格朗日乘数法（条件极值大题）
    ├─ 二重积分直角坐标+极坐标、交换积分次序
    └─ 高数基础全部完成
```

**注意**：这一阶段数学不追求快，目标是进入学习节奏。第一遍不追求难题，计算准确率优先。

## 第二阶段：计组攻坚 + 数学推进（2026.09 - 2026.12）

**目标**：啃下408最硬的骨头，数学进入强化

### 计算机组成原理（主攻）

```
9月：  数据表示 + 运算
      ├─ 原码/反码/补码/移码，IEEE754浮点数
      ├─ 定点运算、ALU
      └─ 你7年没碰硬件，这是最难的部分，慢慢来

10月：存储系统
      ├─ Cache映射（直接/组相联/全相联）
      ├─ 虚拟内存、TLB、页表
      ├─ 这块和OS重叠，配合学效果好
      └─ Cache-主存-辅存三级体系是计组大题核心

11月：指令系统 + CPU
      ├─ 寻址方式、指令格式、CISC vs RISC
      ├─ 数据通路、流水线、冒险处理
      └─ 流水线是计组大题另一个高频点

12月：总线 + I/O + 计组收尾
      ├─ 中断、DMA、总线仲裁
      ├─ 王道计组第一遍完成
      └─ 允许有不懂的，二轮再补
```

**检验标准**：能画出一个完整的CPU数据通路图，讲清楚一条指令的执行过程。

### 数学二

> 详细安排见 [[02-第二阶段-计组-数学]]，教学参考 [[14-科目-数学二]]

```
9月：积分二轮强化 + 微分方程 + 多元微分
    ├─ 不定积分/定积分强化（凑微分提速、分部积分综合、有理函数积分）
    ├─ 微分方程综合题 + 应用题
    ├─ 多元微分强化（复合函数链式法则、隐函数、拉格朗日乘数法）
    └─ 汤家凤强化课 + 1800题强化篇

10月：二重积分二轮 + 线代启动
    ├─ 二重积分强化（交换次序、极坐标、对称性、含绝对值）
    ├─ 线代：行列式（性质消元、特殊行列式）
    ├─ 线代：矩阵（乘法不可交换、逆矩阵、秩、矩阵方程）
    └─ 线代：向量组（线性相关/无关、极大无关组、线性表出）

11月：线代收尾
    ├─ 线性方程组（解的判定、齐次通解、非齐次通解、含参数讨论）
    ├─ 特征值与特征向量（定义、求法、相似对角化）
    └─ 二次型（正交变换化标准型、正定性判定）

12月：线代二轮速过 + 数学二基础收尾
    ├─ 线代全书回顾，大题步骤规范化
    ├─ 高数查漏补缺（薄弱章节针对性补）
    └─ 基础题正确率>80%为过关
```

## 第三阶段：OS + 计网 + 数学收尾（2027.01 - 2027.04）

**目标**：你的优势科目快速收割，数学进入刷题

### 操作系统（快速过，你有体感）

```
1月：进程管理
    ├─ 进程/线程、状态转换、调度算法
    ├─ 同步互斥、信号量、死锁
    └─ 你写Java多线程天天碰这些，学理论很快

2月：内存管理 + 文件系统 + IO
    ├─ 分页/分段/段页式、页面置换算法
    ├─ inode、文件目录、磁盘调度
    └─ Linux天天用，很多概念你有直觉
```

### 计算机网络（最快的一门）

```
3月：体系结构 + 物理层 + 数据链路层
    ├─ OSI/TCPIP模型、CSMA/CD、以太网
    └─ 框架过一遍

4月：网络层 + 传输层 + 应用层
    ├─ IP/ARP/ICMP、RIP/OSPF/BGP
    ├─ TCP/UDP、三次握手四次挥手（天天抓包）
    ├─ HTTP/DNS/SMTP（你的日常工作）
    └─ 最快的一门，你每天在HTTP/TCP上工作
```

### 数学二

> 详细安排见 [[03-第三阶段-OS-计网-数学]]

```
1月：高数强化（极限导数→中值定理→积分→定积分应用+微分方程）
    ├─ 汤家凤强化课 + 1800题强化篇
    └─ 目标：不定积分10道8道以上正确

2月：多元微积分强化 + 线代强化
    ├─ 二重积分综合题（交换次序、极坐标、对称性）
    ├─ 线代强化（行列式→矩阵→向量→方程组→特征值→二次型）
    └─ 正交变换化二次型大题15分钟完成

3-4月：数学二真题一刷（2013-2024，12套）
    ├─ 每周2-3套，限时3小时
    ├─ 错题按章节分类 + 错误类型归类（计算/知识/审题）
    └─ 目标：真题均分>80
```

## 第四阶段：408二轮融合 + 数学刷题（2027.05 - 2027.08）

**目标**：四门课从独立→融合，408真题开始上手

### 408二轮

```
5月：数据结构二轮
    ├─ 算法题专项（每天一道手写代码）
    ├─ 时间复杂度分析强化
    └─ 王道强化课 + 大题专项

6月：计组二轮
    ├─ 一轮模糊的点重点清（Cache、流水线）
    ├─ 大题专项：Cache计算 + 数据通路分析
    └─ 计组是408拉分项，多花时间

7月：OS + 计网二轮
    ├─ 快速过，重点放在408综合题
    ├─ OS+计组的交叉考点（虚拟内存、Cache、TLB联动）
    └─ 这是408的特色：跨科目综合

8月：408真题第一轮（2016-2024，9年真题）
    ├─ 每周2套，做完精析
    ├─ 错题归类：是知识点盲区还是审题问题
    └─ 目标：选择题正确率>70%
```

### 数学二

> 详细安排见 [[04-第四阶段-408二轮]]

```
5月：真题一刷收尾 + 错题专项
    ├─ 错题按15个章节归类，找出最薄弱三章
    ├─ 计算专项训练（每天10道纯计算）
    └─ 薄弱章节正确率从<50%提升到>70%

6月：真题二刷（只做错题+困难题）
    ├─ 二刷还错→标红色（考前必看）
    └─ 目标：一刷错题二刷后正确率>90%

7月：模拟题（张宇8套卷/李林6套卷，每周1-2套）
    ├─ 模拟题比真题难，分数低不慌，目的是暴露盲区
    └─ 目标：模拟题均分>90

8月：真题选填专项 + 回归基础
    ├─ 12套×10道选填=120道，目标选填正确率>85%
    ├─ 公式卡片全部默写
    └─ 目标：选填>48分/56分
```

```

  通勤/午休：30min 背政治/英语单词

  上午：3h 专业课（精力最好时啃408）

  上午：2h 数学
  下午：2h 专业课
```
