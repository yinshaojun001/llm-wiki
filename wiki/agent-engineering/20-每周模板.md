# 每周学习模板

---

## 时间块

```
周一/三/五：
  晚上 21:00-22:30（1.5h）核心学习 — 写代码 / 看文档

周六：
  下午 14:00-18:00（4h）整块时间 — 项目开发 / 深度学习

周日：
  自由（建议 1-2h 整理笔记 + 本周复盘）
```

> 核心原则：工作日晚上保持节奏，周末做需要连续投入的事。**深度工作 1.5h > 碎片 3h**。

---

## 每周 Checklist

### 学之前
- [ ] 回顾上周笔记（5min）
- [ ] 明确本周目标（写下来，不要只在脑子里）

### 学之中
- [ ] 代码写了并跑通了（不跑通 = 没学）
- [ ] 遇到问题先自己 Debug 10min，解不开再搜
- [ ] 每个新概念用 Java 类比写一句话

### 学之后
- [ ] 写一篇本周笔记（学到了什么 + 踩了什么坑 + 下周重点）
- [ ] 代码 push 到 GitHub
- [ ] 在 [[00-总览]] 里打勾本周任务
