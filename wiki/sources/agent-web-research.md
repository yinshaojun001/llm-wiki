# 源：Agent Web 研究集群

**原始位置**：`Obsidian Vault/AI研究 research/agent-web/`
**研究时间**：2026-03-25
**核心主题**：网站在 AI agent 时代的演化方向

---

## 核心结论

Web 正在经历一次范式迁移：从"为人类设计"走向"同时为人类和 AI agent 设计"。这条演化链是：

> [[concepts/seo]] → [[concepts/aeo]] → [[concepts/agent-first]] → [[concepts/agentic-web]]

---

## 关键概念

- [[concepts/aeo]]：Agent Experience Optimization，让 agent 能理解、引用、信任、调用你的网站
- [[concepts/agent-first]]：网站适配 AI agent 的具体策略（语义清晰、结构化数据、API-first）
- [[concepts/agentic-web]]：agent 成为 Web 原生参与者，不只是"使用"Web
- [[concepts/browser-as-runtime]]：浏览器不只显示结果，也能承载 agent 的行为

---

## 核心判断

1. Agent-First 解决**可理解性**，Agentic Web 解决**原生参与性**
2. 机器可读性正在变成网站基础能力（如同移动端适配）
3. AEO 是从 SEO 走向 agent 可见性的过渡优化手段
4. Browser as Runtime 是 Agentic Web 的基础设施层
