## 1. 动态规划（DP）

### 思维框架

```
1. 定义状态：dp[i] 或 dp[i][j] 代表什么？
2. 状态转移方程：dp[i] 怎么从 dp[i-1] 推出来？
3. 初始化：dp[0] 是什么？
4. 遍历顺序：正序还是倒序？
5. 返回值：dp[n] 还是 max(dp[]) ？
```

### 经典：斐波那契

```java
// 递归（有重复计算，O(2^n)）
int fib(int n) {
    if (n <= 1) return n;
    return fib(n - 1) + fib(n - 2);
}

// DP（O(n)）
int fib(int n) {
    if (n <= 1) return n;
    int[] dp = new int[n + 1];
    dp[0] = 0; dp[1] = 1;
    for (int i = 2; i <= n; i++)
        dp[i] = dp[i - 1] + dp[i - 2];
    return dp[n];
}

// 空间优化（O(1)）
int fib(int n) {
    if (n <= 1) return n;
    int prev = 0, curr = 1;
    for (int i = 2; i <= n; i++) {
        int next = prev + curr;
        prev = curr;
        curr = next;
    }
    return curr;
}
```

### 经典：爬楼梯

```java
// dp[i] = dp[i-1] + dp[i-2]
int climbStairs(int n) {
    if (n <= 2) return n;
    int prev = 1, curr = 2;
    for (int i = 3; i <= n; i++) {
        int next = prev + curr;
        prev = curr;
        curr = next;
    }
    return curr;
}
```

### 经典：最长递增子序列（LIS）

```java
// dp[i] = 以 nums[i] 结尾的最长递增子序列长度
int lengthOfLIS(int[] nums) {
    int[] dp = new int[nums.length];
    Arrays.fill(dp, 1);
    int maxLen = 1;
    for (int i = 1; i < nums.length; i++) {
        for (int j = 0; j < i; j++) {
            if (nums[j] < nums[i]) {
                dp[i] = Math.max(dp[i], dp[j] + 1);
            }
        }
        maxLen = Math.max(maxLen, dp[i]);
    }
    return maxLen;
}
// O(n²)，优化到 O(n log n) 用二分查找
```

### 经典：背包问题

```java
// 0-1 背包：每个物品只能选一次
// dp[j] = 容量为 j 时的最大价值
int knapsack(int[] weights, int[] values, int capacity) {
    int[] dp = new int[capacity + 1];
    for (int i = 0; i < weights.length; i++) {
        for (int j = capacity; j >= weights[i]; j--) {  // 倒序遍历！
            dp[j] = Math.max(dp[j], dp[j - weights[i]] + values[i]);
        }
    }
    return dp[capacity];
}

// 完全背包：每个物品可以选无限次
int unboundedKnapsack(int[] weights, int[] values, int capacity) {
    int[] dp = new int[capacity + 1];
    for (int i = 0; i < weights.length; i++) {
        for (int j = weights[i]; j <= capacity; j++) {  // 正序遍历！
            dp[j] = Math.max(dp[j], dp[j - weights[i]] + values[i]);
        }
    }
    return dp[capacity];
}
```

**关键区别**：0-1 背包倒序遍历（每个物品只用一次），完全背包正序遍历（可重复使用）。

### 经典：最长公共子序列（LCS）

```java
int longestCommonSubsequence(String text1, String text2) {
    int m = text1.length(), n = text2.length();
    int[][] dp = new int[m + 1][n + 1];
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (text1.charAt(i - 1) == text2.charAt(j - 1))
                dp[i][j] = dp[i - 1][j - 1] + 1;
            else
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
        }
    }
    return dp[m][n];
}
```

## 2. 贪心算法

**思路**：每一步都选局部最优，期望得到全局最优。

**适用条件**：问题具有贪心选择性质（局部最优能推导全局最优）。

```java
// 经典：跳跃游戏
// 能到达最后一个位置吗？
boolean canJump(int[] nums) {
    int maxReach = 0;
    for (int i = 0; i < nums.length; i++) {
        if (i > maxReach) return false;  // 到不了当前位置
        maxReach = Math.max(maxReach, i + nums[i]);
    }
    return true;
}

// 经典：分发饼干（每个孩子最多一块，求满足的最大孩子数）
int findContentChildren(int[] g, int[] s) {
    Arrays.sort(g);  // 孩子的胃口
    Arrays.sort(s);  // 饼干的大小
    int child = 0, cookie = 0;
    while (child < g.length && cookie < s.length) {
        if (s[cookie] >= g[child]) child++;  // 满足了一个孩子
        cookie++;
    }
    return child;
}

// 经典：区间调度（最多不重叠区间）
int eraseOverlapIntervals(int[][] intervals) {
    Arrays.sort(intervals, Comparator.comparingInt(a -> a[1]));  // 按结束时间排序
    int count = 0, end = Integer.MIN_VALUE;
    for (int[] interval : intervals) {
        if (interval[0] >= end) {
            end = interval[1];  // 保留这个区间
            count++;
        }
    }
    return intervals.length - count;  // 需要移除的数量
}
```

**如何判断能否用贪心**：尝试找反例。如果局部最优不能推导全局最优，就用 DP。

## 3. 回溯算法

**思路**：穷举所有可能，发现走不通就回退。

```java
// 模板
void backtrack(路径, 选择列表) {
    if (满足结束条件) {
        result.add(路径);
        return;
    }
    for (选择 : 选择列表) {
        做选择;
        backtrack(路径, 选择列表);
        撤销选择;  // 回溯
    }
}
```

### 经典：全排列

```java
List<List<Integer>> permute(int[] nums) {
    List<List<Integer>> result = new ArrayList<>();
    backtrack(result, new ArrayList<>(), nums, new boolean[nums.length]);
    return result;
}

void backtrack(List<List<Integer>> result, List<Integer> path, int[] nums, boolean[] used) {
    if (path.size() == nums.length) {
        result.add(new ArrayList<>(path));
        return;
    }
    for (int i = 0; i < nums.length; i++) {
        if (used[i]) continue;
        used[i] = true;
        path.add(nums[i]);
        backtrack(result, path, nums, used);
        path.remove(path.size() - 1);  // 撤销
        used[i] = false;
    }
}
```

### 经典：子集

```java
List<List<Integer>> subsets(int[] nums) {
    List<List<Integer>> result = new ArrayList<>();
    backtrack(result, new ArrayList<>(), nums, 0);
    return result;
}

void backtrack(List<List<Integer>> result, List<Integer> path, int[] nums, int start) {
    result.add(new ArrayList<>(path));  // 每个节点都是一个子集
    for (int i = start; i < nums.length; i++) {
        path.add(nums[i]);
        backtrack(result, path, nums, i + 1);
        path.remove(path.size() - 1);
    }
}
```

### 经典：N 皇后

List<List<String>> solveNQueens(int n) {
    List<List<String>> result = new ArrayList<>();
    char[][] board = new char[n][n];
    for (char[] row : board) Arrays.fill(row, '.');

void backtrack(List<List<String>> result, char[][] board, int row) {
    if (row == board.length) {
        result.add(construct(board));
        return;
    }
    for (int col = 0; col < board.length; col++) {
        if (!isValid(board, row, col)) continue;
        board[row][col] = 'Q';
        backtrack(result, board, row + 1);
        board[row][col] = '.';
    }
}

## 4. 分治算法

**思路**：把问题分成子问题，分别解决，再合并结果。

```java
// 经典：归并排序（已学）
// 经典：快速排序（已学）

// 经典：多数元素（Boyer-Moore 投票算法）
int majorityElement(int[] nums) {
    int candidate = nums[0], count = 1;
    for (int i = 1; i < nums.length; i++) {
        if (count == 0) {
            candidate = nums[i];
            count = 1;
        } else if (nums[i] == candidate) {
            count++;
        } else {
            count--;
        }
    }
    return candidate;
}
```

## 5. 滑动窗口

```java
// 模板
int slidingWindow(String s) {
    int[] window = new int[128];  // 或 HashMap
    int left = 0, right = 0;
    int result = 0;
    while (right < s.length()) {
        char c = s.charAt(right);
        right++;
        // 更新窗口内状态
        window[c]++;

        // 判断是否需要收缩
        while (窗口需要收缩) {
            char d = s.charAt(left);
            left++;
            // 更新窗口内状态
            window[d]--;
        }
        // 更新答案
        result = Math.max(result, right - left);
    }
    return result;
}

// 经典：无重复字符的最长子串
int lengthOfLongestSubstring(String s) {
    Map<Character, Integer> map = new HashMap<>();
    int left = 0, maxLen = 0;
    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);
        if (map.containsKey(c)) {
            left = Math.max(left, map.get(c) + 1);
        }
        map.put(c, right);
        maxLen = Math.max(maxLen, right - left + 1);
    }
    return maxLen;
}
```

## 6. 双指针

```java
// 经典：两数之和（有序数组）
int[] twoSum(int[] numbers, int target) {
    int left = 0, right = numbers.length - 1;
    while (left < right) {
        int sum = numbers[left] + numbers[right];
        if (sum == target) return new int[]{left + 1, right + 1};
        else if (sum < target) left++;
        else right--;
    }
    return new int[]{-1, -1};
}

// 经典：盛最多水的容器
int maxArea(int[] height) {
    int left = 0, right = height.length - 1;
    int maxArea = 0;
    while (left < right) {
        maxArea = Math.max(maxArea,
            Math.min(height[left], height[right]) * (right - left));
        if (height[left] < height[right]) left++;
        else right--;
    }
    return maxArea;
}

// 经典：接雨水
int trap(int[] height) {
    int left = 0, right = height.length - 1;
    int leftMax = 0, rightMax = 0, water = 0;
    while (left < right) {
        if (height[left] < height[right]) {
            if (height[left] >= leftMax) leftMax = height[left];
            else water += leftMax - height[left];
            left++;
        } else {
            if (height[right] >= rightMax) rightMax = height[right];
            else water += rightMax - height[right];
            right--;
        }
    }
    return water;
}
```
