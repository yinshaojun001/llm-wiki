## 1. 排序算法全景

| 算法 | 平均 | 最坏 | 最好 | 空间 | 稳定 |
|---|---|---|---|---|---|
| 冒泡排序 | O(n²) | O(n²) | O(n) | O(1) | 是 |
| 选择排序 | O(n²) | O(n²) | O(n²) | O(1) | 否 |
| 插入排序 | O(n²) | O(n²) | O(n) | O(1) | 是 |
| 希尔排序 | O(n^1.3) | O(n²) | O(n) | O(1) | 否 |
| **归并排序** | O(n log n) | O(n log n) | O(n log n) | O(n) | 是 |
| **快速排序** | O(n log n) | O(n²) | O(n log n) | O(log n) | 否 |
| 堆排序 | O(n log n) | O(n log n) | O(n log n) | O(1) | 否 |
| 计数排序 | O(n+k) | O(n+k) | O(n+k) | O(k) | 是 |
| 基数排序 | O(n×d) | O(n×d) | O(n×d) | O(n+k) | 是 |

**面试重点**：归并排序、快速排序。其他了解即可。

## 2. 快速排序

```java
void quickSort(int[] arr, int lo, int hi) {
    if (lo >= hi) return;
    int pivot = partition(arr, lo, hi);
    quickSort(arr, lo, pivot - 1);
    quickSort(arr, pivot + 1, hi);
}

int partition(int[] arr, int lo, int hi) {
    int pivot = arr[hi];  // 选最后一个做基准
    int i = lo;           // i 指向"小于区"的下一个位置
    for (int j = lo; j < hi; j++) {
        if (arr[j] <= pivot) {
            swap(arr, i, j);
            i++;
        }
    }
    swap(arr, i, hi);
    return i;
}
```

**关键点**：
- partition 是核心：把数组分成"小于 pivot"和"大于 pivot"两部分
- 最坏情况 O(n²)：数组已有序，每次只分出一个元素
- 优化：随机选 pivot / 三数取中

## 3. 归并排序

```java
void mergeSort(int[] arr, int lo, int hi) {
    if (lo >= hi) return;
    int mid = lo + (hi - lo) / 2;
    mergeSort(arr, lo, mid);
    mergeSort(arr, mid + 1, hi);
    merge(arr, lo, mid, hi);
}

void merge(int[] arr, int lo, int mid, int hi) {
    int[] temp = new int[hi - lo + 1];
    int i = lo, j = mid + 1, k = 0;
    while (i <= mid && j <= hi) {
        if (arr[i] <= arr[j]) temp[k++] = arr[i++];
        else temp[k++] = arr[j++];
    }
    while (i <= mid) temp[k++] = arr[i++];
    while (j <= hi) temp[k++] = arr[j++];
    System.arraycopy(temp, 0, arr, lo, temp.length);
}
```

**关键点**：
- 分治思想：先分到最小（单个元素），再合并
- 合并是核心：两个有序数组合并成一个有序数组
- 稳定排序，适合链表排序

## 4. 堆排序

```java
void heapSort(int[] arr) {
    int n = arr.length;
    // 建堆（从最后一个非叶子节点开始）
    for (int i = n / 2 - 1; i >= 0; i--)
        heapify(arr, n, i);
    // 排序：每次把堆顶（最大值）交换到末尾
    for (int i = n - 1; i > 0; i--) {
        swap(arr, 0, i);
        heapify(arr, i, 0);
    }
}

void heapify(int[] arr, int n, int i) {
    int largest = i;
    int left = 2 * i + 1;
    int right = 2 * i + 2;
    if (left < n && arr[left] > arr[largest]) largest = left;
    if (right < n && arr[right] > arr[largest]) largest = right;
    if (largest != i) {
        swap(arr, i, largest);
        heapify(arr, n, largest);
    }
}
```

## 5. 二分查找

### 基本模板

```java
int binarySearch(int[] arr, int target) {
    int lo = 0, hi = arr.length - 1;
    while (lo <= hi) {
        int mid = lo + (hi - lo) / 2;  // 防溢出
        if (arr[mid] == target) return mid;
        else if (arr[mid] < target) lo = mid + 1;
        else hi = mid - 1;
    }
    return -1;
}
```

### 变体一：查找左边界

```java
// 找 target 第一次出现的位置
int leftBound(int[] arr, int target) {
    int lo = 0, hi = arr.length;
    while (lo < hi) {
        int mid = lo + (hi - lo) / 2;
        if (arr[mid] < target) lo = mid + 1;
        else hi = mid;  // 不放过，继续往左找
    }
    if (lo >= arr.length || arr[lo] != target) return -1;
    return lo;
}
```

### 变体二：查找右边界

```java
// 找 target 最后一次出现的位置
int rightBound(int[] arr, int target) {
    int lo = 0, hi = arr.length;
    while (lo < hi) {
        int mid = lo + (hi - lo) / 2;
        if (arr[mid] <= target) lo = mid + 1;  // 等于也往右找
        else hi = mid;
    }
    if (lo - 1 < 0 || arr[lo - 1] != target) return -1;
    return lo - 1;
}
```

### 变体三：旋转数组搜索

```java
// [4,5,6,7,0,1,2] 中找 target
int search(int[] nums, int target) {
    int lo = 0, hi = nums.length - 1;
    while (lo <= hi) {
        int mid = lo + (hi - lo) / 2;
        if (nums[mid] == target) return mid;
        // 左半段有序
        if (nums[lo] <= nums[mid]) {
            if (nums[lo] <= target && target < nums[mid]) hi = mid - 1;
            else lo = mid + 1;
        }
        // 右半段有序
        else {
            if (nums[mid] < target && target <= nums[hi]) lo = mid + 1;
            else hi = mid - 1;
        }
    }
    return -1;
}
```

## 6. 二分查找的应用

```java
// 1. 求 sqrt(x)
int sqrt(int x) {
    long lo = 0, hi = x;
    while (lo <= hi) {
        long mid = lo + (hi - lo) / 2;
        if (mid * mid == x) return (int) mid;
        else if (mid * mid < x) lo = mid + 1;
        else hi = mid - 1;
    }
    return (int) hi;
}
```
