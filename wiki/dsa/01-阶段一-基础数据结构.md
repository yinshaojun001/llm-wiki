# 阶段一：基础数据结构（第 1-3 周）

> 目标：掌握线性数据结构，能分析时间/空间复杂度

---

## 1. 复杂度分析（先学这个）

```java
// O(1) — 常数时间
int getFirst(int[] arr) { return arr[0]; }

// O(n) — 线性时间
int sum(int[] arr) {
    int s = 0;
    for (int x : arr) s += x;
    return s;
}

// O(n²) — 平方时间
void bubbleSort(int[] arr) {
    for (int i = 0; i < arr.length; i++)
        for (int j = 0; j < arr.length; j++)
            if (arr[i] < arr[j]) swap(arr, i, j);
}

// O(log n) — 对数时间
int binarySearch(int[] arr, int target) {
    int lo = 0, hi = arr.length - 1;
    while (lo <= hi) {
        int mid = lo + (hi - lo) / 2;
        if (arr[mid] == target) return mid;
        else if (arr[mid] < target) lo = mid + 1;
        else hi = mid - 1;
    }
    return -1;
}
```

**速记**：O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(2ⁿ)

**关键规则**：
- 只看最高次项：O(3n² + 5n + 100) → O(n²)
- 常数忽略：O(100n) → O(n)
- 嵌套循环相乘：外层 n × 内层 n → O(n²)

## 2. 数组 Array

```java
// Java 数组：固定大小，连续内存
int[] arr = new int[5];
arr[0] = 1;        // O(1) 随机访问
arr.length;         // O(1) 获取长度

// ArrayList：动态数组，自动扩容
List<Integer> list = new ArrayList<>();
list.add(1);        // 均摊 O(1) 追加
list.get(0);        // O(1) 随机访问
list.remove(0);     // O(n) 删除（需要移动元素）
list.size();        // O(1)
```

**核心特性**：随机访问 O(1)，插入/删除 O(n)（需要移动元素）

**Java 工程师已知**：你天天用 ArrayList，这里只是理论化你已有的直觉。

## 3. 链表 LinkedList

```java
// 单链表节点
class ListNode {
    int val;
    ListNode next;
    ListNode(int val) { this.val = val; }
}

// 双链表节点
class DoublyListNode {
    int val;
    DoublyListNode prev, next;
    DoublyListNode(int val) { this.val = val; }
}
```

### 核心操作

```java
// 头插法 O(1)
ListNode head = null;
head = new ListNode(1);  // 新节点
// ... 后续节点

// 遍历 O(n)
ListNode curr = head;
while (curr != null) {
    System.out.println(curr.val);
    curr = curr.next;
}

// 删除节点（给定前驱）O(1)
prev.next = prev.next.next;

// 插入节点（给定位置）O(1)
node.next = prev.next;
prev.next = node;
```

### 常见技巧：虚拟头节点（Dummy Node）

```java
// 删除链表中所有值为 val 的节点
ListNode removeElements(ListNode head, int val) {
    ListNode dummy = new ListNode(0);  // 虚拟头节点
    dummy.next = head;
    ListNode curr = dummy;
    while (curr.next != null) {
        if (curr.next.val == val) {
            curr.next = curr.next.next;
        } else {
            curr = curr.next;
        }
    }
    return dummy.next;  // 真正的头节点
}
```

**为什么要用 dummy**：统一处理头节点被删除的边界情况，避免单独判断。

### 数组 vs 链表

| 操作 | 数组 | 链表 |
|---|---|---|
| 随机访问 | O(1) | O(n) |
| 头部插入 | O(n) | O(1) |
| 尾部插入 | 均摊 O(1) | O(1)（有尾指针） |
| 删除 | O(n) | O(1)（已知前驱） |
| 内存 | 连续，缓存友好 | 分散，缓存不友好 |

## 4. 栈 Stack

```java
// Java 中用 Deque 实现栈（不要用 Stack 类）
Deque<Integer> stack = new ArrayDeque<>();
stack.push(1);      // 入栈 O(1)
stack.push(2);
stack.pop();        // 出栈 O(1)，返回 2
stack.peek();       // 查看栈顶 O(1)，不弹出
stack.isEmpty();    // O(1)
```

**LIFO（后进先出）**：想象一摞盘子，只能从顶部取。

### 经典应用

```java
// 1. 括号匹配
boolean isValid(String s) {
    Deque<Character> stack = new ArrayDeque<>();
    for (char c : s.toCharArray()) {
        if (c == '(') stack.push(')');
        else if (c == '[') stack.push(']');
        else if (c == '{') stack.push('}');
        else if (stack.isEmpty() || stack.pop() != c) return false;
    }
    return stack.isEmpty();
}

// 2. 单调栈（求下一个更大元素）
int[] nextGreaterElement(int[] nums) {
    int[] result = new int[nums.length];
    Arrays.fill(result, -1);
    Deque<Integer> stack = new ArrayDeque<>();  // 存索引
    for (int i = 0; i < nums.length; i++) {
        while (!stack.isEmpty() && nums[stack.peek()] < nums[i]) {
            result[stack.pop()] = nums[i];
        }
        stack.push(i);
    }
    return result;
}
```

## 5. 队列 Queue

```java
// 普通队列
Queue<Integer> queue = new ArrayDeque<>();
queue.offer(1);     // 入队 O(1)
queue.offer(2);
queue.poll();       // 出队 O(1)，返回 1
queue.peek();       // 查看队头 O(1)

// 双端队列
Deque<Integer> deque = new ArrayDeque<>();
deque.offerFirst(1);   // 头部入
deque.offerLast(2);    // 尾部入
deque.pollFirst();     // 头部出
deque.pollLast();      // 尾部出

// 优先队列（堆）
Queue<Integer> pq = new PriorityQueue<>();           // 小顶堆
Queue<Integer> maxPq = new PriorityQueue<>(Comparator.reverseOrder()); // 大顶堆
pq.offer(3);  pq.offer(1);  pq.offer(2);
pq.poll();    // 返回 1（最小的）
```

**FIFO（先进先出）**：想象排队买奶茶。

### 经典应用：滑动窗口最大值

```java
// 用单调递减双端队列
int[] maxSlidingWindow(int[] nums, int k) {
    Deque<Integer> deque = new ArrayDeque<>();  // 存索引
    int[] result = new int[nums.length - k + 1];
    for (int i = 0; i < nums.length; i++) {
        // 移除超出窗口的元素
        while (!deque.isEmpty() && deque.peekFirst() < i - k + 1)
            deque.pollFirst();
        // 维护单调递减
        while (!deque.isEmpty() && nums[deque.peekLast()] < nums[i])
            deque.pollLast();
        deque.offerLast(i);
        if (i >= k - 1) result[i - k + 1] = nums[deque.peekFirst()];
    }
    return result;
}
```
