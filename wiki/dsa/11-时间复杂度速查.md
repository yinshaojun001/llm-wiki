# 时间复杂度速查表

---

## 数据结构操作复杂度

| 数据结构 | 访问 | 查找 | 插入 | 删除 | 空间 |
|---|---|---|---|---|---|
| 数组 | O(1) | O(n) | O(n) | O(n) | O(n) |
| 动态数组 | O(1) | O(n) | 均摊 O(1) | O(n) | O(n) |
| 链表 | O(n) | O(n) | O(1) | O(1) | O(n) |
| 栈 | O(n) | O(n) | O(1) | O(1) | O(n) |
| 队列 | O(n) | O(n) | O(1) | O(1) | O(n) |
| 哈希表 | — | O(1) 均摊 | O(1) 均摊 | O(1) 均摊 | O(n) |
| BST（平衡） | — | O(log n) | O(log n) | O(log n) | O(n) |
| 堆 | — | O(n) | O(log n) | O(log n) | O(n) |
| Trie | — | O(L) | O(L) | O(L) | O(N×L) |

> L = 字符串长度，N = 字符串数量

## 排序算法复杂度

| 算法 | 平均 | 最坏 | 最好 | 空间 | 稳定 |
|---|---|---|---|---|---|
| 冒泡 | O(n²) | O(n²) | O(n) | O(1) | 是 |
| 选择 | O(n²) | O(n²) | O(n²) | O(1) | 否 |
| 插入 | O(n²) | O(n²) | O(n) | O(1) | 是 |
| 归并 | O(n log n) | O(n log n) | O(n log n) | O(n) | 是 |
| 快排 | O(n log n) | O(n²) | O(n log n) | O(log n) | 否 |
| 堆排 | O(n log n) | O(n log n) | O(n log n) | O(1) | 否 |
| 计数 | O(n+k) | O(n+k) | O(n+k) | O(k) | 是 |
| 基数 | O(n×d) | O(n×d) | O(n×d) | O(n+k) | 是 |

## 常见复杂度量级

| 复杂度 | n=10 | n=100 | n=1000 | n=10⁶ | 可接受？ |
|---|---|---|---|---|---|
| O(1) | 1 | 1 | 1 | 1 | 总是 |
| O(log n) | 3 | 7 | 10 | 20 | 总是 |
| O(n) | 10 | 100 | 1000 | 10⁶ | 是 |
| O(n log n) | 33 | 664 | 10⁴ | 2×10⁷ | 是 |
| O(n²) | 100 | 10⁴ | 10⁶ | 10¹² | n ≤ 10⁴ |
| O(n³) | 1000 | 10⁶ | 10⁹ | 10¹⁸ | n ≤ 500 |
| O(2ⁿ) | 1024 | 10³⁰ | — | — | n ≤ 20 |
| O(n!) | 3.6M | — | — | — | n ≤ 10 |

**面试经验**：一般要求 O(n) 或 O(n log n)。O(n²) 在 n ≤ 1000 时可接受。
