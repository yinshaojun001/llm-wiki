# 数据结构与算法：从入门到精通

> 目标：系统掌握 DSA，能独立分析和解决中等难度算法题
> 对象：7 年 Java 后端工程师，有基本编程能力但算法训练不足
> 时间：约 16 周（每周 8-10 小时）

---

## 为什么学 DSA

- 面试硬门槛：大厂/外企算法轮占比 40-60%
- 工程直觉：理解 HashMap 为什么 O(1)、数据库索引为什么用 B+ 树
- Agent 工程需要：向量检索、图遍历、最短路径都是算法
- 你已有 Java 基础，数据结构的"骨架"你已经懂，现在补"肌肉记忆"

## 学习原则

1. **先暴力后优化**：每道题先用最笨的方法做出来，再优化
2. **画图先行**：链表、树、图必须画，别在脑子里跑
3. **手写伪代码**：先写思路再写代码，不要边想边敲
4. **模板优先**：二分、BFS、DFS、DP 都有模板，先背模板再变形
5. **限时训练**：中等题 30 分钟，超时就看题解，不要死磕
