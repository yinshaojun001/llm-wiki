# Agent-First

> 网站不只要对人类用户友好，也要对 AI agent 友好。

## 核心目标

让 AI agent 能够：
- 理解内容语义
- 提取事实
- 判断可信度
- 调用网站能力
- 代表用户完成任务

## 四大特征

1. **语义清晰**：内容结构化，意图明确
2. **[[concepts/structured-data]]**：机器可读的标记
3. **API-first**：能力通过 API 暴露
4. **信息一致**：不同入口的信息不矛盾

## 与 Agentic Web 的关系

Agent-First 是**当下可做的事**（适配），[[concepts/agentic-web]] 是**未来的形态**（融合）。Agent-First 是通向 Agentic Web 的过渡阶段。
