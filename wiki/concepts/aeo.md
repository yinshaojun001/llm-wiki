# AEO

**Agent Experience Optimization**，Agent 体验优化。

> [[concepts/aeo]] 解决的是"让 agent 愿意选择你"。

## 与 SEO 的区别

| | SEO | AEO |
|---|---|---|
| 目标 | 人类用户点击 | Agent 引用和调用 |
| 优化指标 | 收录、排名、点击率 | 理解度、引用率、信任度、可调用性 |
| 时代 | 搜索引擎时代 | AI agent 时代 |

## 四个优化维度

1. **可理解**：agent 能准确解析内容含义
2. **可引用**：内容格式适合被 agent 截取引用
3. **可信任**：信息准确一致，有来源依据
4. **可调用**：网站能力通过 API 暴露给 agent
