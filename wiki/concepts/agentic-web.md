# Agentic Web

> agent 不只是访问 Web，而是成为 Web 的原生参与者。

## 核心特征

- Agent 参与界面层（不只是"读取"）
- 交互是连续的、有状态的
- UI 是可生成的（不依赖人类预设的 DOM 结构）
- [[concepts/browser-as-runtime]] 是其基础设施层

## 与 [[concepts/agent-first]] 的区别

| | Agent-First | Agentic Web |
|---|---|---|
| 解决的问题 | 可理解性 | 原生参与性 |
| 视角 | 网站适配 agent | Agent 成为 Web 一部分 |
| 阶段 | 当下过渡期 | 未来演化方向 |
