# SEO（搜索引擎优化）

> 为人类使用的搜索引擎优化内容可见性，是 Agent 时代之前 Web 可发现性的主导范式。

## 核心逻辑

通过关键词、外链、内容质量、技术结构等手段，让网页在 Google/Bing 等搜索引擎排名靠前，从而获取人类用户的流量。

## 在演化链中的位置

SEO 解决的是"让人类找到你"。随着 AI agent 成为信息获取的新入口，SEO 的作用边界正在收窄：

`**SEO** → [[concepts/aeo]] → [[concepts/agent-first]] → [[concepts/agentic-web]]`

- SEO 优化目标：搜索引擎爬虫 + 人类点击
- [[concepts/aeo]] 优化目标：AI agent 的引用、信任与调用

## 局限性（Agent 时代）

- 关键词密度对 LLM 无效
- PageRank 类算法不影响 Agent 的内容选取
- 传统 SEO 元数据（meta description 等）对 Agent 可见性贡献有限
